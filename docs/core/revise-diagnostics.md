# 改写与诊断

## 适用场景
- 剧情跑偏
- 人设失真
- 节奏拖沓
- 章节太水
- 重复推进
- 伏笔拖欠
- 进度提前

## 六类高频问题
1. **主线跑偏**：连续多章答不出“这章如何服务当前阶段目标”
2. **角色失真**：角色行为不再服从既有欲望、恐惧、关系位置
3. **承诺拖欠**：只提不还，读者一直等不到兑现或补偿
4. **节奏拖沓**：有字数没变化，有场面没结果
5. **超纲抢跑**：提前兑现不该兑现的节点
6. **套路重复**：近几章总是同一种小赢、同一种口头威慑、同一种虚钩子

## 诊断顺序
1. 先看本章功能有没有丢
2. 再看变化有没有落到局面
3. 再看是不是和近几章重复
4. 再看是否误伤后续主线
5. 最后才做句子级修辞处理

## 默认修复策略
- 跑偏：回到阶段目标与卷目标
- 失真：回看人物动机与关系位置
- 太水：压缩无效铺垫，补结果变化
- 太平：补冲突、代价、选择压力
- 太早：延后兑现，只做预热或偏移补偿
- 太像：强制换结果类型或换冲突结构
