# Genre Asset Index

Genre material is an on-demand asset layer. It should not replace the core writing workflow.

## Order

1. Start with `docs/core/open-book.md`, `docs/core/write-workflow.md`, and `docs/core/revise-diagnostics.md`.
2. Load genre assets only when the genre question is specific.
3. Read only the 1-2 most relevant files at a time.
