# 末世资料入口

> 这页是当前默认的末世资料入口。先走 `open / write / quality / status` 主链，只在题材问题明确时按需补读这里。

## 使用顺序

1. 先看 `docs/core/open-book.md`，确认题材、卖点和阶段目标。
2. 再看 `assets/genres/04-moshi.md`，建立末世题材的基础骨架。
3. 需要细化时，再按下面几类资料挑 1-2 份补读，不要整包灌入主链。

## 资料分层

### 题材基线

- `assets/genres/04-moshi.md`
- `assets/genres/06-moshi-tunhuo.md`
- `assets/genres/apocalypse/pack.md`

### 章节卡与推进节点

- `references/moshi-chapter-card-library.md`
- `references/moshi-bastion-governance-chapter-cards.md`

适用场景：
- 需要补“这一章靠什么活下去”
- 需要安排搜物资、守据点、改秩序的推进节点

### 据点、秩序与治理

- `references/moshi-bastion-and-order-library.md`

适用场景：
- 需要把安全区、避难所、队伍秩序写成真实冲突
- 需要安排资源分配、治理规则和派系摩擦

### 人物学习

- `references/moshi-character-learning.md`

适用场景：
- 需要拉开主角、队友、上位者、掠夺者的生存立场
- 需要把人物弧线写成资源压力和道德选择

### 负面模式

- `references/moshi-negative-patterns.md`

适用场景：
- 只有逃命和打怪，没有局势升级
- 囤货像流水账，没有资源重量
- 金手指白给，没有代价或后账

### 卷路与长线升级

- `references/moshi-volume-route-library.md`

适用场景：
- 需要规划灾变升级、据点扩张和文明回潮
- 需要安排“守住什么、放弃什么、重建什么”的长线节奏

## 一句话判断

- 如果你在解决“末世题材像不像”，先看题材基线。
- 如果你在解决“这一章怎么活下来”，先看章节卡。
- 如果你在解决“据点秩序怎么立住”，先看据点与治理资料。
- 如果你在解决“为什么越写越空”，先看负面模式。
- 如果你在解决“长线怎么撑住”，先看卷路。
