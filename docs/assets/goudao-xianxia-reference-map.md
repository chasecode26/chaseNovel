# 仙侠苟道资料入口

> 这页是当前默认的仙侠苟道资料入口。先走 `open / write / quality / status` 主链，只在题材问题明确时按需补读这里。

## 使用顺序

1. 先看 `docs/core/open-book.md`，确认题材、卖点和阶段目标。
2. 再看 `assets/genres/05-xianxia-goudao.md`，建立苟道流的基础题材骨架。
3. 需要细化时，再按下面四类资料挑 1-2 份补读，不要整包灌入主链。

## 资料分层

### 题材基线

- `assets/genres/05-xianxia-goudao.md`
- `assets/genres/goudao-xianxia/pack.md`

### 章节卡与推进节点

- `references/goudao-chapter-card-library.md`

适用场景：
- 需要补“这一章靠什么推进”
- 需要给隐忍、试探、试错和反杀节奏找落点

### 人物学习与站位变化

- `references/goudao-character-learning.md`

适用场景：
- 需要拉开主角、同门、宗门、猎杀者的风险感
- 需要把“苟”写成策略，不是胆小拖延

### 负面模式

- `references/goudao-negative-patterns.md`

适用场景：
- 一直躲不出手，成长长期停滞
- 底牌只藏不用，安全区永远不失效
- 机缘白给，没有代价和后账

### 卷路与长线升级

- `references/goudao-volume-route-library.md`

适用场景：
- 需要规划资源积累、境界推进和暗线反杀
- 需要安排“藏多久、露多少、什么时候必须出手”的长线节奏

## 一句话判断

- 如果你在解决“苟道题材像不像”，先看题材基线。
- 如果你在解决“这一章怎么推进”，先看章节卡。
- 如果你在解决“主角为什么这样苟”，先看人物学习。
- 如果你在解决“为什么越写越闷”，先看负面模式。
- 如果你在解决“长线升级怎么撑住”，先看卷路。
