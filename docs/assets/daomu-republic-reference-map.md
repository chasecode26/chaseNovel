# 盗墓 / 民国奇诡资料入口

> 这页是当前默认的盗墓 / 民国奇诡资料入口。先走 `open / write / quality / status` 主链，只在题材问题明确时按需补读这里。

## 使用顺序

1. 先看 `docs/core/open-book.md`，确认题材、卖点和阶段目标。
2. 再看 `assets/genres/10-daomu-minguo.md`，建立盗墓 / 民国奇诡题材的基础骨架。
3. 需要细化时，再按下面几类资料挑 1-2 份补读，不要整包灌入主链。
