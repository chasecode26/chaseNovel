# 都市系统流资料入口

> 这页是当前默认的都市系统流资料入口。先走 `open / write / quality / status` 主链，只在题材问题明确时按需补读这里。

## 使用顺序

1. 先看 `docs/core/open-book.md`，确认开书题材、卖点和章功能。
2. 再看 `assets/genres/02-dushi-xitongliu.md`，建立都市系统流的基础题材骨架。
3. 需要细化时，再按下面四类资料挑 1-2 份补读，不要整包灌入主链。

## 资料分层

### 题材基线

- `assets/genres/02-dushi-xitongliu.md`
- `assets/genres/dushi-system/pack.md`

### 章节卡与推进节点

- `references/dushi-chapter-card-library.md`

适用场景：
- 需要补“这一章该交什么结果”
- 需要给系统奖励、现实回响、反制节奏找落点

### 人物学习与关系推进

- `references/dushi-character-learning.md`

适用场景：
- 需要拉开主角、对手、跟班、围观者的站位差
- 需要把“打脸”写成关系和资源重排，而不是一群人震惊

### 负面模式

- `references/dushi-negative-patterns.md`

适用场景：
- 系统播报太多，现实反应太少
- 配角只会震惊，冲突没有真实代价
- 装穷打脸反复循环，结果没有升级

### 卷路与长线升级

- `references/dushi-volume-route-library.md`

适用场景：
- 需要规划 10-30 章的升级节奏
- 需要安排阶段性 payoff、资源膨胀和更高层对手

## 一句话判断

- 如果你在解决“题材像不像”，先看题材基线。
- 如果你在解决“这一章交什么”，先看章节卡。
- 如果你在解决“人物为什么会变位”，先看人物学习。
- 如果你在解决“为什么越写越空”，先看负面模式。
- 如果你在解决“后面几十章怎么升”，先看卷路。
