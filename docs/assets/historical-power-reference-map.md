# 历史权谋资料入口

> 这页是当前默认的历史权谋资料入口。先走 `open / write / quality / status` 主链，只在题材问题明确时按需补读这里。

## 使用顺序

1. 先看 `docs/core/open-book.md`，确认题材、卖点和阶段目标。
2. 再看 `assets/genres/09-lishi-quanmou.md`，建立历史权谋的基础题材骨架。
3. 需要细化时，再按下面几类资料挑 1-2 份补读，不要整包灌入主链。

## 资料分层

### 题材基线

- `assets/genres/09-lishi-quanmou.md`
- `assets/genres/historical-power/pack.md`

### 章节卡与推进节点

- `references/historical-chapter-card-library.md`
- `references/historical-military-political-chapter-card-library.md`

适用场景：
- 需要补“这一章靠什么推进”
- 需要安排朝堂、军政、边关交错推进的节奏

### 人物、关系与阵营变化

- `references/historical-character-learning.md`
- `references/historical-relationship-card-deck.md`

适用场景：
- 需要拉开皇帝、宗室、外戚、军府、边将的立场差
- 需要把人物关系写成利益和站位变化，而不只是对话斗嘴

### 军令、后效与中盘推进

- `references/historical-command-and-aftermath-library.md`
- `references/historical-midgame-learning.md`

适用场景：
- 需要把军令执行、战后后账、财政代价写实
- 需要规划扩盘、结盟、整军、夺权等中盘阶段

### 负面模式

- `references/historical-negative-patterns.md`

适用场景：
- 权谋只靠嘴，局势没有真实代价
- 战争只有气氛，没有后勤、财政和军心支点
- 配角像木偶，只会配合主角聪明

### 卷路与长线升级

- `references/historical-volume-route-library.md`

适用场景：
- 需要规划从地方到朝堂、从军府到国运的长线升级
- 需要安排阶段性夺权、清算和治国压力

## 一句话判断

- 如果你在解决“题材像不像”，先看题材基线。
- 如果你在解决“这一章怎么推进”，先看章节卡。
- 如果你在解决“谁会站哪边”，先看人物和关系资料。
- 如果你在解决“为什么局势不落地”，先看军令后效和负面模式。
- 如果你在解决“长线怎么撑住”，先看卷路。
