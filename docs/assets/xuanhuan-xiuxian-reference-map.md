# 玄幻 / 修仙资料入口

> 这页是当前默认的玄幻 / 修仙资料入口。先走 `open / write / quality / status` 主链，只在题材问题明确时按需补读这里。

## 使用顺序

1. 先看 `docs/core/open-book.md`，确认题材、卖点和阶段目标。
2. 再看 `assets/genres/01-xuanhuan-xiuxian.md`，建立玄幻 / 修仙题材的基础骨架。
3. 需要细化时，先看 `assets/genres/xuanhuan-xiuxian/pack.md`，再按需挑 1-2 份子风格或通用资料补读。

## 一句话判断

- 如果你在解决“玄幻 / 修仙题材像不像”，先看题材基线。
- 如果你在解决“这本该走哪种修仙子风格”，先看子风格入口。
- 如果你在解决“开篇怎么起、战斗怎么写、文风怎么落”，再看通用技法。
