# 种田资料入口

> 这页是当前默认的种田 / 基建资料入口。先走 `open / write / quality / status` 主链，只在题材问题明确时按需补读这里。

## 使用顺序

1. 先看 `docs/core/open-book.md`，确认题材、卖点和阶段目标。
2. 再看 `assets/genres/03-zhongtianwen.md`，建立种田 / 基建题材的基础骨架。
3. 需要细化时，再按下面几类资料挑 1-2 份补读，不要整包灌入主链。

## 资料分层

### 题材基线

- `assets/genres/03-zhongtianwen.md`
- `assets/genres/farming/pack.md`

### 章节卡与阶段推进

- `references/zhongtian-chapter-card-library.md`

适用场景：
- 需要补“这一章具体交付了什么改善”
- 需要安排农事、家计、邻里压力和阶段收获的推进节点

### 人物学习与关系经营

- `references/zhongtian-character-learning.md`

适用场景：
- 需要拉开主角、家人、乡邻、族老、商贩的立场差
- 需要把关系变化写成资源合作、利益分配和情感回流

### 负面模式

- `references/zhongtian-negative-patterns.md`

适用场景：
- 只有种地记账，没有生活秩序升级
- 收成在涨，但人物关系和村庄格局不动
- 金手指一路白给，没有劳动、时间或社会代价

### 卷路与长线升级

- `references/zhongtian-volume-route-library.md`

适用场景：
- 需要规划从活下去到家业扩张、产业升级、乡里立足的长线节奏
- 需要安排“守住什么、扩大什么、换来什么麻烦”的阶段跃迁

## 一句话判断

- 如果你在解决“种田题材像不像”，先看题材基线。
- 如果你在解决“这一章改善了什么”，先看章节卡。
- 如果你在解决“人和家为什么愿意跟主角一起干”，先看人物学习。
- 如果你在解决“为什么越写越像流水账”，先看负面模式。
- 如果你在解决“长线经营怎么撑住”，先看卷路。
