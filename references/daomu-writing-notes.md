# 盗墓 / 摸金 / 民国奇诡题材笔记

这份文档只沉淀当前最常用的盗墓题材写法，不展开大而全设定。

重点解决：

- 盗墓题材容易只剩“下墓打怪”
- 民国背景容易只剩皮，不进局
- 术语、机关、禁忌很多，但剧情推进弱

---

## 一、题材的真实推进器是什么

盗墓文真正抓人的不是墓本身，而是四层东西同时推进：

1. 墓里的危险
2. 墓外的势力
3. 主角身上的旧债、旧案或旧诅咒
4. 每次下墓都带出来的新情报、新仇家或新利益

如果只有第一层，就会迅速塌成一次次副本重复。

---

## 二、最稳的开篇抓手

### 1. 先异常，再入局

- 先给不该出现的文物、残影、尸变、旧地图或诡异求救
- 再让主角不得不出手
- 不要一开始就讲完整门派史

### 2. 先低位，再露手艺

- 主角最好先处于落魄、被追债、被追杀、被看轻、快饿死的状态
- 第一章就要给一次“手艺是真的有用”的小验证
- 例如闻土、辨位、破机关、识伪货、看风水

### 3. 先人，再墓

- 先让读者记住主角为什么冒这个险
- 墓只是局，不是故事全部
- 如果主角的命、债、仇、诅咒都不成立，下墓就只剩流程

---

## 三、民国摸金为什么容易成立

民国好用，不只是因为气氛。

它天然自带四种冲突：

- 黑帮 / 军阀 / 租界 / 日本势力互卡
- 法外灰区，很多事能做又做不干净
- 文物、情报、军火、黄金都能和墓产生关系
- 新旧秩序并存，人物站队和身份伪装都更好写

所以民国摸金不要只写墓下机关，也要写：

- 谁想拿墓里的东西
- 谁想借墓里的线索换地位
- 谁想让主角死在墓里

---

## 四、常用卖点组合

### 1. 祖传手艺 + 落魄入局

- 适合快起手
- 读者容易接受主角有真本事

### 2. 诅咒 / 寿命压力 + 必须下墓

- 适合挂长期主线
- 比单纯求财更能撑长线

### 3. 文物异能 / 残影读取 + 历史谜案

- 适合把研究感和爽点绑在一起
- 但不能每次都靠金手指直接给答案

### 4. 摸金 + 商战 / 势力经营

- 适合长期连载
- 每次下墓不只为了宝物，还为了地位、资源、人脉和后续盘面

---

## 五、最常见失误

| 失误 | 表现 | 修正方向 |
|------|------|---------|
| 只会写副本 | 每个墓都像新地图刷怪 | 每次下墓必须带出势力变化、人物后账或主线信息 |
| 术语堆积 | 黑话、风水词、门派词一起上 | 第一次出现只解释当前场景需要理解的部分 |
| 主角像导游 | 一直负责讲解设定 | 让手艺通过动作、判断和结果体现 |
| 气氛空转 | 阴风、诡异、寒意反复堆 | 用机关、尸体位置、脚印、气味、声音落地 |
| 墓外没戏 | 出墓后剧情失速 | 墓外必须有抢货、灭口、交易、站队、追查 |

---

## 六、章节设计的最小要求

一章盗墓文，至少要落住下面三项里的两项：

- 新危险
- 新信息
- 新后账

最好再带一项：

- 新人物判断

如果一章只有“走路、看墓、感叹、解释”，默认偏空。

---

## 七、术语使用原则

- 摸金符、分金定穴、寻龙定位、尸变、机关、禁忌，这些都该少量多次地带出来
- 不要把术语解释写成百科词条
- 同一个术语保持叫法稳定，必要时放进 `worldbuilding-index.md`
- 真实考据词和虚构词混用时，优先保证当前章节可读性

---

## 八、章尾钩子推荐

| 类型 | 示例 |
|------|------|
| 墓里还有活人 | 墓门快合上时，里面有人敲了三下 |
| 带出来的东西不干净 | 那枚玉蝉到了地面，居然还是温的 |
| 墓外势力堵门 | 他刚出巷口，就看到日本特高课的车停在对面 |
| 旧案再起 | 图纸背面那行字，正是他父亲的笔迹 |
| 规矩被破 | 黑驴蹄子掉地的那一刻，师父脸色变了 |

---

## 九、一句话原则

**盗墓文不是“进墓拿宝”这么简单，而是拿命去碰秘密，再把秘密带回人间变成更大的局。**
