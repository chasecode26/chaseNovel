# 本地下载语料优先级地图

这份文档不是讲题材原则，而是回答一个更实际的问题：

**`D:\git\sonovel-windows\SoNovel\downloads` 里已经有一批小说，接下来先读哪本，分别拿来学什么。**

原则只有三条：

- 只提炼结构、节奏、人物关系、结果路线、卷级推进，不搬运原文
- 优先读能直接服务当前五条主战题材的样本
- 每次只带着问题去读，不整本平推

---

## 一、最高优先级样本

这些样本与当前技能主战题材直接相关，且已经能看出明显可提炼价值。

### 1. 历史权谋 / 边境战争

#### `庆余年(猫腻).txt`

适合学习：

- 开篇如何先给“异常人生”再给“高压追杀”
- 婴儿视角如何不幼稚，反而制造世界谜团
- 如何用一次截杀把身世、势力、秘密、长期承诺同时挂上去
- 如何让“保护者”与“掌权者”一出场就形成气场差与潜台词冲突

不该只学：

- 不要只学穿越壳
- 要学“开篇一场事同时挂四条线”的密度

#### `赘婿(愤怒的香蕉).txt`

适合学习：

- 如何先用现代失败者的终局感，反衬古代新身份的再来一次
- 如何让主角一进新世界就先落在关系局里，而不是先炫能力
- 如何把“赘婿身份低位”“家族内斗”“女主未归”“主角不想入局”一起压进起步局
- 如何把商战、家族斗、时代大势拆成不同层级，先近后远铺开

不该只学：

- 不要只盯着装逼反打
- 要学“低位身份 + 高认知视角 + 复杂关系局”的切入法

#### `晚明(柯山梦).txt`

适合学习：

- 如何用荒诞入局降低历史门槛，再迅速转向生存压力
- 如何让“双主角能力差”自然形成分工，不是复制粘贴双男主
- 如何让“衣食住行都没着落”先于“大历史抉择”发生
- 如何把穿越后的第一目标压缩成非常现实的求生链

不该只学：

- 不要只学搞笑和裸穿噱头
- 要学“边地乱世先求活，再谈改局”的推进顺序

### 2. 末世囤货

#### `全球冰封：我打造了末日安全屋(记忆的海).txt`

适合学习：

- 如何开篇先给惨死记忆，立刻建立复仇与生存双驱动
- 如何在一章内完成“末世强度说明 + 重生确认 + 异能到账 + 囤货任务开启”
- 如何让读者迅速接受主角变冷、变狠、变不圣母
- 如何把“囤货”写成具体计划，而不是情绪喊话

不该只学：

- 不要只学堆物资名录
- 要学“复仇名单 + 生存缺口 + 资源方案”三件事同时推进

### 3. 仙侠苟道

#### `凡人修仙传！(小疯子的风).txt`

适合学习：

- 如何先把主角按在低处，让贫寒、孤独、务实先立住
- 如何用生活困苦和外界异常，慢慢把“凡人视角进入超凡”拉开
- 如何让主角先像一个会过日子的人，再谈修行
- 如何把“世界可能要变”的信号压在日常缝隙里，而不是上来就天降机缘

不该只学：

- 不要只学苦
- 要学“低资源生存感 + 异象渐近 + 人先成立”的慢起手

### 4. 都市系统流 / 权力上升线

#### `权欲：从乡镇到省委大院(任羡鱼).txt`

适合学习：

- 如何开篇直接给人生抉择，不做长铺垫
- 如何让“重生”立刻绑定明确仇恨与晋升目标
- 如何通过家庭处境、大伯态度、五万门槛，迅速立住主角的低位与现实阻力
- 如何把“体制权力”从抽象概念落到一个看得见的羞辱场景

不该只学：

- 不要只学重生打脸
- 要学“低位受压 -> 看见规则 -> 决心入局”的因果链

## 二、第二梯队样本

这些样本价值高，但当前技能还没系统拆到对应资产，可以按需抽读。

### `三国之凤舞九天(诸葛不要太亮).txt`

适合补：

- 边境战争推进
- 将门 / 部将 / 阵营关系
- 战时成长与统兵线

建议用途：

- 后续可补一份“将领关系与战场调度”专项

### `神话版三国(坟土荒草).txt`

适合补：

- 大地图长线推进
- 群像势力切换
- 战略级爽点如何避免塌成数值播报

建议用途：

- 后续可补一份“大战略卷级换挡”专项

### `将夜(猫腻).txt`

适合补：

- 世界观压迫感
- 少年成长中的硬目标与情绪目标并行
- 关系线里的长期承诺

建议用途：

- 后续可补一份“高压世界观下的人物驱动”专项

### `第九特区(伪戒).txt`

适合补：

- 灰度秩序
- 末世后秩序感 / 底层生存感
- 团队与地盘线的连续升级

建议用途：

- 如果要把末世囤货往“组织化生存”方向再推，可以重点抽

## 三、按题材的实际抽读顺序

### 如果当前重点是历史权谋 / 边境战争

1. `庆余年(猫腻).txt`
2. `赘婿(愤怒的香蕉).txt`
3. `晚明(柯山梦).txt`
4. `三国之凤舞九天(诸葛不要太亮).txt`
5. `神话版三国(坟土荒草).txt`

优先提炼：

- 开篇如何挂长期承诺
- 家族 / 阵营 / 部将关系如何动态变化
- 战争不是打仗流水，而是资源、命令、违令、背锅、后账

### 如果当前重点是末世囤货

1. `全球冰封：我打造了末日安全屋(记忆的海).txt`
2. `第九特区(伪戒).txt`

优先提炼：

- 囤货如何尽快过渡到据点、团队、规则、秩序
- 邻里冲突之后，怎么继续往势力冲突升级

### 如果当前重点是仙侠苟道

1. `凡人修仙传！(小疯子的风).txt`
2. `将夜(猫腻).txt`

优先提炼：

- 苟不是缩着不动，而是低资源下的判断、伪装、避险、借力
- 长期目标必须早挂，不然容易塌成日常受苦账

### 如果当前重点是都市系统流 / 权力上升线

1. `权欲：从乡镇到省委大院(任羡鱼).txt`

优先提炼：

- 起步低位怎么立
- 规则门槛怎么落地
- 主角为什么一定要往上爬

## 四、下一轮最值得补的资产

如果继续按本地语料深挖，优先顺序建议如下：

### 1. 历史线补“将领关系与战场后账库”

原因：

- 当前历史线已有开篇、中盘、人物、章卡、整卷、反套路
- 但还可以再往“命令链、军功链、后勤链、违令后账”深挖一层

### 2. 末世线补“据点升级与秩序重建库”

原因：

- 当前末世线已经有囤货、团队、章卡、反套路
- 下一层应该是从个人堡垒过渡到组织秩序

### 3. 苟道线补“风险判断与借力关系库”

原因：

- 当前苟道线有卷路、人物、章卡、反套路
- 但“为什么这次该苟、该借、该杀、该退”的判断模型还能再精炼

### 4. 都市系统流补“规则反制与身份揭面库”

原因：

- 现在已有基础资产
- 但还能把“会议桌、考核、站队、后台、关系后账”拆成可直接套用的章卡群

## 五、实际使用方法

不要一上来就去读整本。

推荐动作：

1. 先确定这次要补哪条题材线
2. 再确定这次卡的是开篇、中盘、人物、章卡还是整卷
3. 带着问题去抽本地样本的前几章、转折章、卷末章
4. 把学到的内容沉淀成路线、章卡、关系、反套路，不记原文

一句话原则：

**本地下载语料不是拿来“抄像”，而是拿来拆“为什么这一步会让人想继续看”。**
