# references

`references/` now keeps only task contracts. It no longer stores genre notes, old-path shims, or migration history.

## Kept

- `references/contracts/`: task contracts for launch, write, continue, revise, state/rhythm, promise/repeat, and research material.

## Default entry points

- Core workflow: `docs/core/`
- Genre assets: `docs/assets/` and `assets/genres/`
- Templates: `templates/core/`, `templates/launch/`, `templates/review/`

Put new reusable material under `assets/` or `docs/assets/`; do not grow root `references/` into a second asset library.
