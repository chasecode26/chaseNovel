# 风格档案约束

本文件用于建立题材口吻与单书口吻的双层真相源。

## 双层结构

### 题材口吻

描述该题材普遍允许的语言强度、旁白浓度、句式节拍。

### 单书口吻

描述当前作品自己的叙述声音、主角感知习惯、禁写表达、偏好修辞。

## 题材默认建议

### 玄幻 / 修仙

- 句子可短促有力
- 少写空灵大词，重结果、重异象、重代价
- 旁白可以冷硬，但别悬浮

### 都市 / 系统

- 更口语、更直接
- 少端着，少作者腔
- 反馈必须清晰可见

### 悬疑 / 推理

- 控制解释欲
- 优先给线索、异常、矛盾
- 少提前替读者定义气氛

### 言情 / 情感

- 情绪可以浓，但必须落在人际互动、动作细节、误读与期待上
- 少写空洞唯美，多写关系变化

## 单书档案至少包含

- 题材
- 叙述视角
- 句式节拍偏好
- 允许的旁白浓度
- 禁写表达
- 慎用表达
- 常见场景写法偏好
- 角色语言差异

## 使用规则

1. 题材口吻用于兜底。
2. 单书口吻高于题材口吻。
3. 改写时如果出现冲突，优先服从单书档案。
4. 如果单书档案缺失，必须先补最小档案，再做大规模语言统一。
