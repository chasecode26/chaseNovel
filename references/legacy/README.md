# legacy references

`references/legacy/` 是历史兼容隔离区，不是默认阅读入口。

这里的文件只承担两件事：

- 保留旧路径，避免历史链接直接失效
- 把维护者导向当前主入口

默认请先看：

1. `docs/core/open-book.md`
2. `docs/core/write-workflow.md`
3. `docs/core/revise-diagnostics.md`
4. `docs/core/status-workflow.md`
5. `docs/core/task-contracts.md`
6. `docs/assets/genre-index.md`

除非你在追溯历史链接或兼容旧项目，否则不要从这里继续下钻。
