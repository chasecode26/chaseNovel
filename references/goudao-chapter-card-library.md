# 仙侠苟道章节卡样例库

---

## 1. 避险入局章

```markdown
# 第 5 章写作方案
- 本章功能：立安全边界
- 本章冲突：外门筛选、资质低、稍有出头就会被盯
- 本章结果类型：保命失机型
- 章尾钩子类型：选择逼近型
- 本章章尾钩子：一个低调但危险的机会摆在主角面前，不拿亏，拿了可能暴露
```

## 2. 小机缘到手章

```markdown
# 第 13 章写作方案
- 本章功能：证明苟有收益
- 本章冲突：资源有限、同门窥伺、动静不能大
- 本章结果类型：资源到手型
- 章尾钩子类型：信息反转型
- 本章章尾钩子：主角刚拿到小机缘，就发现原来这东西有人提前做了标记
```

## 3. 伪装潜伏章

```markdown
# 第 24 章写作方案
- 本章功能：隐藏底牌
- 本章冲突：必须出手，但又不能让别人知道是自己
- 本章结果类型：赢局面丢关系型
- 章尾钩子类型：关系突变型
- 本章章尾钩子：某个原本无关的人开始怀疑主角不对劲
```

## 4. 机缘取舍章

```markdown
# 第 37 章写作方案
- 本章功能：让苟道真正成立
- 本章冲突：大机缘当前，但进去要冒高风险
- 本章结果类型：保命失机型
- 章尾钩子类型：欲望升级型
- 本章章尾钩子：主角虽然没去，却看到进去的人带出了让他也动心的东西
```

## 5. 安全区失效章

```markdown
# 第 51 章写作方案
- 本章功能：换挡
- 本章冲突：原本安全的身份、地点或规则已经不能再保命
- 本章结果类型：失手受损型
- 章尾钩子类型：危机压顶型
- 本章章尾钩子：主角必须在短时间内换身份、换地方或换路线
```

---

一句话原则：

**苟道章卡的核心不是“更强了”，而是“活下来了，但活法必须变”。**
