# 写作文风执行基线

这份文档不是“文学理论”，而是把成熟网文提示词改写成 `chaseNovel` 可执行规则后的文风主控件。

适用场景：

- 你已经知道这章写什么，但总写成发白、发硬、发同质
- 你不想再被“默认大白话”压成一个口气
- 你想保留题材文气、角色身份感、场面余味，但又不想掉进故作高深

## 一、总原则

1. **清晰可读，不等于口水化。**
   - 读者要能判明谁在做事、为什么做、结果变了什么。
   - 但表达不必统一扁平，不必所有题材都像同一个现代人口语。
2. **文气服务题材、身份和场景，不服务“显得会写”。**
   - 历史、权谋、古风、慢热情绪段可以更收、更稳、更有余味。
   - 都市系统、末世压迫、快节奏兑现段优先利落、直接、少绕。
3. **含蓄和发雾不是一回事。**
   - 允许潜台词、试探、让步、遮掩、欲言又止。
   - 不允许读者看完整段，仍判断不出谁要什么、谁怕什么、谁占上风。
4. **对白是行动，不是信息广播。**
   - 每轮对话至少推动立场、关系、决策、风险、交易中的一项。
5. **诗句、典故、文学作品只能做加分项。**
   - 只能增强质感，不能替代推进。
   - 一旦压过动作、结果、冲突、兑现，直接删。

## 二、执行判断

### 1. 什么叫“写对了”

- 读者能迅速进入场面
- 能感到人物身份差异和题材气味
- 能判断人物真实意图，而不是只听到一堆狠话
- 场面中的动作、站位、物件、环境、代价在持续工作
- 句子有层次，但不逼读者停下来翻译

### 2. 什么叫“写歪了”

- 一味追求直白，把题材口吻、人物身份、场景余味全部抹平
- 一味追求文气，把信息写成雾，把判断写成谜
- 全员一个腔，主角、配角、朝臣、兵卒、商贩说话都像同一个人
- 对话全是“态度”，没有筹码、没有交换、没有结果
- 环境只负责装饰，不提供限制、机会、代价或转场钩子

## 三、对白规则

### 必须做到

- 每轮对白至少承担一个功能：推进决策、交换信息、拉扯关系、暴露立场、制造后果。
- 台词要能听出身份、阅历、怕点、筹码和处境。
- 纯对白默认不够，至少补一类东西：站位、视线、动作、打断、停顿、环境反馈。
- 潜台词可以有，但读者要能判明“谁在试探 / 谁在敷衍 / 谁在威胁 / 谁在求稳 / 谁在让价”。

### 直接删改信号

- 只剩狠话对轰，局面不动
- 只在搬信息，不在争利益
- 看着有劲，离开上下文就不知道这句到底在干什么
- 为了显得老练，把正常交流写得人人都像谜语人
- 用“主口”“主杀招”“有鬼”这类作者脑内简称顶掉关键战术信息

## 四、旁白与叙述规则

### 必须做到

- 单场景默认第三人称限知，跟一个焦点人物走。
- 判断句优先顺序：事实 -> 人物判断 -> 后果。
- 抽象感受必须有动作、环境、物件、身体反应或资源变化托底。
- 每 `2-4` 段最好至少推进一次压力、信息、关系或结果中的一项。
- 涉及军报、敌情、推理、权谋判断时，必须直接说清“对方要干什么 / 从哪动手 / 为什么这样判断”。

### 直接删改信号

- 旁白替人物喊口号
- 句子只剩气氛词，没有承载动作
- 一段删掉后，剧情和关系毫无变化
- 连续几句都在解释“为什么重要”，却没有场面本身
- 结论说了，事实没说；判断很硬，读者却不知道问题到底是什么

## 五、引文与文气边界

- 历史、权谋、古风、书信、祭文、碑刻、慢热情绪段，可低频到中频使用诗句、典故、文学化句式。
- 都市系统、末世压迫、追逃激战、爽点兑现段默认慎用或不用。
- 引文必须满足三条：
  - 符合角色身份
  - 不拖慢反馈
  - 删掉后不会发现原来它在替剧情干活

## 六、最短执行法

写前先定 5 件事：

- 本场景跟谁的视角
- 本场景核心压力是什么
- 这场对话真正争的是什么
- 这一场语言应更利落，还是更克制
- 这一场是否允许引文、典故或较重文气

写后只查 5 件事：

- 是否清晰，但没有被写平
- 是否有文气，但没有写雾
- 是否有对白，但不是空对白
- 是否有气氛，但不是假气氛
- 是否有风格，但没有压过推进

一句话总纲：

**让读者既看得懂，又愿意继续看；既能感到题材味，又不会被作者腔挡在门外。**

## 战争文直白化基线（新增）

用于边关战争、守城、军报、调兵、烽燧、粮道等场景。

### 默认写法

- 先写现场：哪里在打、谁在抬人、哪条线在乱
- 再写判断：主角为什么觉得敌人另有打法
- 最后写命令：谁去守哪、谁去搬药、谁去埋伏
- 第一次出现的术语，当句或下一句翻成人话

### 禁止默认写法

- 不要先甩一个术语，再让读者靠后文猜意思
- 不要用“像是懂的人才懂”的黑话制造高级感
- 不要把军情写成谜语
- 不要把命令写成口号
- 不要用作者概括腔替代现场依据和后果

### 战时对白默认模板

- 军报：`前头哪边出了什么事，对面现在怎么动`
- 判断：`若我们把人调过去，对面会从哪边上来`
- 命令：`谁带多少人去哪里，带什么东西，守什么点`

### 快速替换表

- `主口` -> `正面城墙`
- `主杀招` -> `真正要上的地方`
- `压不住` -> `守不住`
- `接刃` -> `真正打起来`
- `错灯` -> `看错军情，把消息报错`
- `有鬼` -> `他们的人埋伏在那边`
- `这一口/那一口` -> `这一阵/这次进攻`
