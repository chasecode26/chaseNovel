# 末世囤货章节卡样例库

只给能直接套用的章卡。

---

## 1. 极寒抢购章

```markdown
# 第 6 章写作方案
- 本章功能：囤货爽点兑现 + 立时间差优势
- 本章冲突：商超限购、运输不稳、仇人也在抢
- 本章结果类型：资源到手型
- 章尾钩子类型：危机压顶型
- 本章章尾钩子：主角刚把最后一批柴油运回，气温骤降速度突然提前
```

## 2. 安全屋漏洞章

```markdown
# 第 14 章写作方案
- 本章功能：从“囤到”切换为“守住”
- 本章冲突：供电系统、制冷链、邻里窥探同时出问题
- 本章结果类型：保命失机型
- 章尾钩子类型：信息反转型
- 本章章尾钩子：主角查出不是设备坏了，而是有人在外面试探他的能耗曲线
```

## 3. 邻居清算章

```markdown
# 第 21 章写作方案
- 本章功能：处理近身威胁
- 本章冲突：旧日受助者反咬、楼内势力抱团
- 本章结果类型：小胜型
- 章尾钩子类型：关系突变型
- 本章章尾钩子：楼里刚压住，外面真正的大势力已经开始打听主角名字
```

## 4. 团队分裂章

```markdown
# 第 33 章写作方案
- 本章功能：人物弧推进 + 提高秩序代价
- 本章冲突：资源不足，团队成员对分配标准出现裂缝
- 本章结果类型：赢局面丢关系型
- 章尾钩子类型：选择逼近型
- 本章章尾钩子：主角必须决定是保规则，还是保一个关键成员
```

## 5. 区域势力试探章

```markdown
# 第 48 章写作方案
- 本章功能：把私人求生升级成区域博弈
- 本章冲突：外部势力试探性交易，实则摸底据点
- 本章结果类型：翻盘型
- 章尾钩子类型：资源争夺型
- 本章章尾钩子：交易完成后，对方在撤离路线故意留下标记
```

---

一句话原则：

**末世章卡不能只写“拿到了什么”，必须写“拿到以后谁会来抢、哪里会失衡、下一步怎么更难活”。**
