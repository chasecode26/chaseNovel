# 落地写作与常识校验契约

> 这份契约用于防止长篇连载写作中反复出现“看似有气势、实际看不懂”“作者替读者总结”“战场/制度/生活常识想当然”“命令句无法执行”的问题。写章、改章、校稿都要先按这里过一遍。

## 1. 问题总表

### 1.1 抽象词承载剧情

高危写法：用“局势、位置、站位、态度、绑定、牵制、太冷太热、名分、压住、稳住、风口、咬住、挨刀、棋子、看重、弃子”等词替代具体事实。

处理规则：
- 正文里出现抽象词时，先问：谁看见了什么？谁要做什么？谁会因此受损？
- 军政判断必须写成“事实 -> 判断 -> 后果”。
- 权谋关系必须落到“谁递帖子、谁收礼、谁问婚期、谁在朝会上能追问、谁会因此不敢动手”。
- 战争压力必须落到“哪扇门、哪段墙、多少粮、多少箭、多少人、敌人从哪里来”。

### 1.2 作者总结替代场面

高危写法：
- “这不是京里来的官话。”
- “这是守门能用的法子。”
- “满堂都明白了。”
- “真正要命的是……”

处理规则：
- 能用动作和反应让读者看懂，就不要替读者总结。
- 章尾不要升华，章尾要给危险、动作、下一步。
- 若一句话删掉后信息没有少，只剩“味道”，默认删或改成动作/后果。

### 1.3 常识想当然

必须校验这些常识：
- 军事：弓箭距离、骑兵夜袭、城门防守、拒马、火油、粮道、哨骑、攻城动作。
- 制度：皇子钦差、监军、守备、兵部文书、驿站、粮仓钥匙、兵册粮册城防册交接、城门口令符验。
- 生活：冻伤、饥饿、伤兵行动能力、民夫说话方式、霉粮、草料掺水对马的影响。
- 动作：人不能随手接高速箭；应写刀背磕偏、盾挡、低头避开、擦身而过。
- 时间：三日前、五日粮、今日剩粮、路程、传信速度必须能算通。
- 物理：湿草、门闩、门缝、钩索、火箭、滚水、石灰都要符合实际使用场景。

不确定时：先查资料或改成更稳的动作，不能硬写。

### 1.4 指令句不可执行

高危写法：
- “照这个方向射。”
- “先射火。”
- “先吊兵器。”
- “他能说。”
- “南门能撑。”

处理规则：
- 军令、调防、审问、查账必须让执行者听完就知道做什么。
- 指令句优先包含：谁去做、做什么、对谁/哪里做、何时做、做到什么程度。
- 临战口令短，但不能省掉目标。

可用改法：
- 不写“照这个方向射”，写“看谁举火箭，就射谁；钩索一绷紧，就射绳后面的人”。
- 不写“先射火”，写“先射举火箭的人”。
- 不写“先吊兵器”，写“让城下的人把弓刀放进筐里，城上收走兵器，再吊人”。
- 不写“南门能撑”，写“南门先按原令守，不从北门抽人”，并给依据。
- 不写“他能说”，写“他说的是敌骑怎么骗城里调兵，让哨骑把话说完”。

### 1.5 记录感过重

高危写法：何川、马承福、文书反复“记、写、抄、记录”，导致正文像流水账。

处理规则：
- 同一场戏最多突出一次记录动作。
- 只有三类信息允许明写入册：回传京城证据、入城后对照兵粮城防数字、影响生死路线的异常。
- 普通信号改成角色动作：看清人、问清路、对上证物、留心来人。
- 如果写“记”，必须让读者知道这条记录后续有什么用。

### 1.6 题材语境错位

古代军政边关题材，不要写成现代办公腔或现代分析腔。

替换原则：
- 不写“账上还有三十七个”，写“兵册上还写着三十七个哨骑，营里实到二十一人，能骑马出城的只剩十六人”。
- 不写“粮车怎么补”，写“下一趟粮从哪座仓出，走哪条路，谁押送，几日能到”。
- 不写“沈家和三皇子绑定”，写“东宫、四皇子府、兵部、户部会借贺喜问沈家态度；沈家若不接，旁人就知道三皇子身后无人追问”。

### 1.7 对话机械问答

高危写法：父女、君臣、上下级一问一答，像作者拿角色做讲解。

处理规则：
- 对话必须带关系压力：试探、压制、避责、求证、让步、威胁、给台阶。
- 不要连续三轮“问什么/答什么”；中间要有动作、停顿、旁人反应、现实物件介入。
- 角色说话不要替作者解释设定；让角色为了自己的利益说话。

## 2. 写章前硬 Gate

起稿前必须回答：
1. 本章最值钱的一场戏是什么？读者这一章看什么热闹？
2. 本章涉及的时间、地点、人数、伤势、粮草、兵器、命令是否接上上一章？
3. 本章敌我双方知道什么、不知道什么？消息是怎么送到的？
4. 本章要用到的制度、军务、生活细节有没有常识风险？不确定的是否已查证或改成稳妥写法？
5. 本章军令/审问/调防是否每一句都能执行？
6. 本章反派处在哪一层？本章只抓当前层证据，不提前掀最终幕后。
7. 本章章尾钩子指向下一步危险或行动，不做命运总结。

## 3. 写后自检 Gate

交稿前逐项扫：
- 是否出现“局势、位置、态度、绑定、名分、压住、风口、挨刀”等抽象词承载关键剧情？有则改成具体人、物、动作、后果。
- 是否出现“真正要命的是、这说明、这意味着、不是……而是……”等作者总结？有则删或改成场面反应。
- 是否有角色知道自己不该知道的事？
- 是否有常识硬伤：人徒手接箭、弓箭距离乱写、粮草时间算错、城门流程乱写、官职权责乱写？
- 是否有“先射火、先吊兵器、照这个方向射”这种执行者会愣的命令？
- 是否有一场戏反复写“记录/记下/写下”？
- 是否有现代办公腔、分析词直接进入古代正文？

## 4. 反派与伏笔阶梯

长篇连载不能每次出事都像孤立事件。写章前必须加载项目里的：
- `00_memory/foreshadowing.md`
- `00_memory/timeline.md`
- `00_memory/state.md`
- 如存在：`00_memory/volume1_antagonist_ladder.md`、`00_memory/*antagonist*.md`、`00_memory/*ladder*.md`

规则：
- 当前章只回收当前层级能回收的证据。
- 低层反派要有可见动作和可审证据；高层反派只留下手法、利益方向和遮掩痕迹。
- 暗杀、投毒、换防、断粮这类情节必须回写伏笔用途，否则就是白写。

## 5. 常见反例对照

| 问题句 | 问题 | 改法方向 |
| --- | --- | --- |
| 先砍马，再割麻袋 | 目标和动作别扭 | 先冲拉车的马下手，马一倒粮车走不动，再割粮袋把粮撒进雪里 |
| 账上还有三十七个 | “账上”不合题材语境 | 兵册上还写着三十七个，营里实到多少，能上城/出城多少 |
| 他敢射皇子 | 尊卑和常识风险 | 改成箭本来射向旁人，守兵慌乱松弦，主角用刀背磕偏 |
| 是他们还要认人 | 含义不清 | 把昨夜守南门的人叫来，当面认换防兵的脸 |
| 烧开门缝，铁钩拖门 | 物理动作不稳 | 火烧门缝逼守军退，钩索拖拒马，内应拔门闩/拆门梁 |
| 南门能撑 | 判断过满 | 南门先按原令守，不从北门抽人，并说明依据 |
| 先射火 | 目标不清 | 先射举火箭的人 |
| 先吊兵器 | 执行动作不清 | 让城下的人把弓刀放进筐里，城上收走兵器，再吊人 |
| 这不是京里来的官话 | 作者总结 | 让老兵直接照法子传令，读者自己看出有用 |
