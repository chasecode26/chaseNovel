# 末世囤货与仙侠苟道钩子 / 结果路线笔记

这份文档解决两个题材的章节收束问题：

- 末世囤货
- 仙侠苟道 / 凡人流

核心目标：

**不要让章节总是“又囤一点 / 又稳一点 / 又差一点”。**

---

## 一、末世囤货

### 结果路线优先级

#### 1. `gain`

适合：

- 前期快速立住囤货爽点
- 安全屋、物资、渠道、空间能力到手

使用原则：

- 资源到手后，必须改变下一章行动条件

不要这样写：

- 只是又加库存，但局势没有任何变化

#### 2. `small_win`

适合：

- 第一波抢资源
- 试探性清算
- 局部压住邻居、仇人、竞争者

使用原则：

- 连续两章内可以用，三章开始就要换

#### 3. `win_with_cost`

适合：

- 中盘开始提升压力
- 抢到资源但暴露位置
- 守住据点但消耗过大

这是末世题材最该多用的结果类型。

#### 4. `survive_but_miss`

适合：

- 灾难突袭
- 避开正面团灭
- 不得不放弃一批物资、一个盟友、一次机会

这类结果最适合防止“末世像经营游戏”。

#### 5. `loss`

适合：

- 主角体系第一次真的出血
- 安全感已经建立后需要重新压迫

使用原则：

- 不是单纯吓唬，而是让据点、信任、资源、时间真的受损

### 末世钩子优先级

优先挂：

- 下一波灾难倒计时
- 据点漏洞暴露
- 新势力盯上主角
- 囤到的关键资源有副作用 / 争夺风险
- 团队内部出现价值冲突

少挂：

- 单纯“还有更多物资”
- 单纯“他还有底牌没出”

### 末世换挡建议

如果最近几章都是囤货、清点、升级，立刻换成：

1. 资源到手，但人心松了
2. 据点稳了，但外面有人盯上
3. 保住命了，但错过关键窗口
4. 打退敌人了，但声名扩散

## 二、仙侠苟道 / 凡人流

### 结果路线优先级

#### 1. `survive_but_miss`

最适合苟道。

适合：

- 避开杀劫
- 放弃机缘
- 活下来但没吃满收益

核心作用：

- 强化“苟是一种理性取舍”

#### 2. `gain`

适合：

- 安全收益
- 隐蔽资源
- 不显眼的小机缘

使用原则：

- 得到的必须能增强“更稳地活下去”，而不是只是数值涨了

#### 3. `win_with_cost`

适合：

- 迫不得已出手
- 化解风险，但暴露了一层底牌
- 拿到机会，但沾了因果、结了仇

这是苟道中盘防腻的关键结果类型。

#### 4. `small_win`

适合：

- 前期证明主角的判断和谨慎是有效的

使用原则：

- 只能短用，不能连刷

#### 5. `loss`

适合：

- 主角误判
- 苟法被破
- 安全边界失效

使用原则：

- 一旦用了，就要推动主角调整策略，而不是原地恢复

### 苟道钩子优先级

优先挂：

- 风险感知升级
- 身份 / 行踪差点暴露
- 更大机缘出现，但去不去都危险
- 旧有安全区不再安全
- 某个关系人把主角拖向高风险场域

少挂：

- “他感觉天道有异”
- “前方似乎有秘密”

这类太虚，不够支撑苟道的下一章行动。

### 苟道换挡建议

如果最近几章总是“稳稳拿小好处”，立刻换成：

1. 活下来，但错过更大机缘
2. 拿到好处，但身份多了一层风险
3. 避险成功，但安全区失效
4. 压住敌人，但暴露了判断方式

---

## 三、两类题材的共同禁忌

### 1. 不要假结果

结果必须改变局面，不能只是换一种情绪描述。

### 2. 不要空钩子

章尾必须挂具体问题、具体倒计时、具体代价。

### 3. 不要连续同类反馈

- 末世不能连着几章都只是“物资到账”
- 苟道不能连着几章都只是“稳了一点”

### 4. 不要只有爽，没有后账

末世和苟道都很怕短期反馈太强、长期压力断掉。

---

## 四、直接套用的结果换挡表

### 末世囤货

- `gain -> win_with_cost`
- `small_win -> survive_but_miss`
- `gain -> loss`

### 仙侠苟道

- `small_win -> survive_but_miss`
- `gain -> win_with_cost`
- `survive_but_miss -> gain`

原则：

- 不一定越来越惨，但一定越来越不一样

---

## 五、直接套用的章尾钩子

### 末世囤货

- “物资拿到了，但制冷 / 供电 / 运输链断了一环”
- “据点守住了，但主角名字被新的敌对势力记住了”
- “这次活下来了，但下一轮极寒 / 暴雪 / 尸潮只剩多少小时”

### 仙侠苟道

- “这次避开了，但有人开始怀疑主角不该活下来”
- “这次没暴露，但原来的安全位置已经不能再待”
- “机缘出现了，但去了可能暴露，不去就要被别人拿走”

这些钩子能直接转成下一章行动，不会只停在氛围。
