# 历史权谋 / 边境战争的人物关系卡组

这份文档只做一件事：

**把历史权谋、边境战争里最常见、最值钱的人物关系，拆成可速查的“关系变质卡组”。**

适用题材：

- 历史权谋
- 边境战争
- 军政争霸
- 智勇双全爽文

配合文件：

- `references/historical-character-learning.md`
- `references/historical-command-and-aftermath-library.md`
- `references/historical-military-political-chapter-card-library.md`

## 一、怎么使用这份卡组

每张卡只回答四件事：

1. 这对关系最常见的起手姿态是什么
2. 中盘最值钱的变质点是什么
3. 最适合用什么事件触发
4. 章尾最适合挂什么后果

使用原则：

- 一章只重点推进一到两组关系
- 不要把所有关系都同时写“更近了”或“更远了”
- 最值钱的不是关系好坏，而是角色因此改变了立场和行为

## 二、核心关系卡组

### 1. 主角 vs 皇帝 / 君上

起手姿态：

- 试用
- 观察
- 借力

中盘最值钱的变质点：

- 君上第一次把信任变成牵制
- 主角第一次发现“被重用”本身就是陷阱

最适合的触发事件：

- 大捷封赏
- 越级请命
- 拒绝一项看似美好的任命
- 监军回京密报

最适合挂的章尾后果：

- 赏赐下来了，但附带分兵、调防、迁职
- 君上没发怒，反而又给了主角一次更危险的机会

一句话用法：

**这组关系不要写成“圣恩”或“君臣相得”，而要写成“越有用，越难自由”。**

---

### 2. 主角 vs 监军

起手姿态：

- 互看不顺眼
- 表面同殿，实际抢链

中盘最值钱的变质点：

- 监军承认主角会打，但开始更细地卡军令、军报、军功
- 主角发现监军不是只想害他，而是想把胜利写成自己可控的胜利

最适合的触发事件：

- 临战夺令
- 单独上奏
- 捷报署名之争
- 封赏使到来前后的情报截流

最适合挂的章尾后果：

- 监军输了这一场，但抢到了向上解释的权
- 主角稳住战场，却失去一段朝堂缓冲

一句话用法：

**监军不是纯坏，而是“也想赢，但不能让你赢得脱离控制”。**

---

### 3. 主角 vs 宿将 / 老资格将领

起手姿态：

- 轻视
- 观望
- 不服年轻主帅

中盘最值钱的变质点：

- 从看不起变成承认本事
- 从承认本事变成担心主角挤掉自己位置

最适合的触发事件：

- 一次真战果
- 一次公开定功
- 一次违令处罚
- 一次老将判断失误被主角补锅

最适合挂的章尾后果：

- 宿将当面服了，背后却开始联络旧系
- 宿将没有反，但再也不肯把全部底牌交给主角

一句话用法：

**宿将线最值钱的不是“终于服了”，而是“服了，但开始防了”。**

---

### 4. 主角 vs 心腹先锋 / 猛将

起手姿态：

- 敬服主角
- 愿意替主角拼命

中盘最值钱的变质点：

- 心腹太能打，开始不守规矩
- 主角越信任他，越容易被他的冒进和忠诚反噬

最适合的触发事件：

- 违令追敌
- 擅改部署
- 代主角背锅
- 被外部拉拢或被封赏单独点名

最适合挂的章尾后果：

- 心腹被罚却不服
- 心腹表面请罪，背后已被别人盯上

一句话用法：

**这组关系要写“制度和情分冲突”，不要只写兄弟义气。**

---

### 5. 主角 vs 粮官 / 转运使 / 掌书记

起手姿态：

- 事务性配合
- 表面恭顺

中盘最值钱的变质点：

- 主角发现这批人不是中立技术官，而是后方权力链的手脚
- 对方也意识到主角不再是只会打仗的粗人

最适合的触发事件：

- 粮册差额
- 调拨迟滞
- 军资错签
- 抚恤名册出错

最适合挂的章尾后果：

- 查出来的不只是账错，而是人心有别
- 主角想动一个账房，结果牵出朝中一线

一句话用法：

**后勤官线不是为了补设定，而是为了把战争真正接进政治。**

---

### 6. 主角 vs 地方官 / 郡守 / 豪族

起手姿态：

- 表面合作
- 实际互相嫌麻烦

中盘最值钱的变质点：

- 地方官发现主角不只是借路，而是真要改地盘秩序
- 主角发现地方官并非怕敌，而是更怕军队坐大

最适合的触发事件：

- 征粮征马
- 仓位争夺
- 城防征调
- 抚恤、迁民、屯田安排

最适合挂的章尾后果：

- 地方官桌上输了，桌下联名上奏
- 豪族暂时让了，却开始断后路、传流言、收买部曲

一句话用法：

**这组关系要写“军和地争的是资源与秩序”，不是单纯互骂。**

---

### 7. 主角 vs 台谏 / 清流 / 老臣

起手姿态：

- 怀疑
- 防备
- 看不起武人做大

中盘最值钱的变质点：

- 他们承认主角有用，但必须防止主角变成制度漏洞

最适合的触发事件：

- 封赏过重
- 越权征调
- 军法处置引争议
- 地方民怨被借题发挥

最适合挂的章尾后果：

- 朝堂上没人能直接杀你，但每个人都能慢慢削你
- 主角赢了一场辩论，却失去了中立名声

一句话用法：

**台谏线不是单纯唱反调，而是制度成本的人形化。**

---

### 8. 主角 vs 盟友统帅

起手姿态：

- 临时合作
- 各怀算盘

中盘最值钱的变质点：

- 对方希望你赢，但不能让你独赢
- 主角发现盟友最大的价值不是托底，而是拿来借势和背锅

最适合的触发事件：

- 战前约兵
- 粮草互借
- 战后分功
- 城池归属、俘虏归属、战利归属争议

最适合挂的章尾后果：

- 盟友没背刺，但已经不再同步信息
- 盟友嘴上还叫盟友，实际已经进入互卡阶段

一句话用法：

**盟友线最值钱的不是翻脸，而是“还没翻脸时已经不能真信”。**

---

### 9. 主角 vs 底层士卒 / 亲兵

起手姿态：

- 看战功
- 看敢不敢冲

中盘最值钱的变质点：

- 从敬主角勇，变成信主角能活人
- 一旦主角在抚恤、换防、赏罚上失手，军心会先裂

最适合的触发事件：

- 伤亡过大
- 抚恤失序
- 违令处罚
- 主角亲自担责或亲自认账

最适合挂的章尾后果：

- 军心暂稳，但底下第一次有人开始怀念旧主帅
- 亲兵还跟着主角，普通军士却开始分层

一句话用法：

**底层军心不能只靠热血，要靠让人看见跟着你更有活路。**

## 三、最好用的关系三连动

如果想让一章不单薄，可以优先用这几组三连动：

### 1. 皇帝 -> 监军 -> 主角

适合写：

- 封赏藏刀
- 越级掣肘
- 试用转忌惮

### 2. 主角 -> 心腹先锋 -> 宿将

适合写：

- 违令有功
- 定功定责
- 主帅位确立

### 3. 主角 -> 粮官 -> 地方官

适合写：

- 后勤断供
- 军地互卡
- 战争转政治

### 4. 主角 -> 台谏 -> 皇帝

适合写：

- 功高震主
- 名望反噬
- 表面加封、实则收权

## 四、快速选卡法

如果这章重点是：

- 谁想控你：先看 `主角 vs 皇帝 / 监军 / 台谏`
- 谁不服你：先看 `主角 vs 宿将 / 心腹先锋`
- 谁在卡你资源：先看 `主角 vs 粮官 / 地方官`
- 谁不能真信：先看 `主角 vs 盟友统帅`
- 谁决定你还能不能统军：先看 `主角 vs 底层士卒 / 亲兵`

## 五、快检清单

写完关系推进章后，快速检查：

1. 本章推进的是哪一组关系卡？
2. 这组关系是因为具体事件变了，还是只因为说了几句狠话？
3. 关系变了之后，谁的资源、位置、选择跟着变了？
4. 章尾挂的是人心后果、军令后果，还是只挂抽象局势？
5. 这组关系下一次再出现时，是否已经不能回到原位？

如果 1-5 里有两条答不上来，这组关系还没真正活起来。
