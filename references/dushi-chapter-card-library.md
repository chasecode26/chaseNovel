# 都市系统流章节卡样例库

---

## 1. 会议打脸章

```markdown
# 第 8 章写作方案
- 本章功能：兑现打脸 + 拿资源
- 本章冲突：上司设坑、同事看戏、客户难缠
- 本章结果类型：资源到手型
- 章尾钩子类型：结果未揭晓型
- 本章章尾钩子：刚拿下客户，系统刷新一个更高风险的连锁任务
```

## 2. 身份揭面章

```markdown
# 第 17 章写作方案
- 本章功能：身份反差兑现
- 本章冲突：反派逼主角亮底牌
- 本章结果类型：翻盘型
- 章尾钩子类型：关系突变型
- 本章章尾钩子：真正的大人物开始主动接触主角
```

## 3. 规则反制章

```markdown
# 第 29 章写作方案
- 本章功能：从打个人升级到打规则
- 本章冲突：反派靠制度、资本或关系卡位
- 本章结果类型：赢局面丢关系型
- 章尾钩子类型：信息反转型
- 本章章尾钩子：主角发现幕后真正出手的不是台前反派
```

## 4. 上层试探章

```markdown
# 第 42 章写作方案
- 本章功能：把主角推入更高圈层
- 本章冲突：高层给机会，也给掣肘
- 本章结果类型：保命失机型
- 章尾钩子类型：选择逼近型
- 本章章尾钩子：主角必须决定要不要正式站队
```

---

一句话原则：

**都市系统流章卡不能只写“这次又打脸”，要写“打脸之后身份、资源、规则哪个变了”。**
