# 飞书网文写作知识库摘要 (Nines & 九格智能)
此文件提取自飞书系列写作宝典，作为 `chaseNovel` 的核心知识补充字典。

## 一、 核心架构引擎

### 1. 七步循环法（长篇网文爽点逻辑 - 百万字基石）
网文的本质是“情绪价值”，通过循环建立期待与满足：
- **展示特质**：第一章立刻亮出主角的独特底牌/金手指/金刚钻。
- **信息差构建**：主角扮猪吃虎，配角狗眼看人低，利用认知不对称拉升期待。
- **伏笔预埋**：在当前日常或低压剧情中，顺手预埋3-5章后大高潮的线索。
- **侧面烘托**：严禁直接描写主角多强，必须通过路人的倒吸凉气、反派的战栗来侧写。
- **意外转折**：不要平铺直叙赢，要三波三折（即将赢 → 突发变故看似要输 → 绝地大翻盘）。
- **具象收割**：升级必须可视化，如“经验条拉满”、“获取SSS级神器”、“被全城权贵跪拜”。

### 2. 雪花构思法 & 许荣哲七步法
- **雪花法**：一句话核心梗 → 一段话扩写 → 一页大纲 → 详细角色卡 → 卷纲 → 章纲，层层分型，避免AI直接写长篇导致的幻觉与逻辑断层。
- **许荣哲七步法**：目标 → 阻碍 → 努力 → 结果 → 意外 → 转折 → 结局。适用于单章起承转合。

## 二、 黄金开篇公式（前三章决胜）

### 1. 中长篇“抓人”开局
- **核心结构**：`可视化收获的实体化展示` + `埋设新势力伏笔` + `悬念句式结尾`
- **三大空间锚定法**：
  - 远景/环境：奠定基调（冷雨、废墟、霓虹灯）。
  - 中景/动态：人物行动轨迹。
  - 特写/细节：微表情、随身关键道具（如带血的玉佩）。
- **避坑红线**：禁止AI味说教；禁止开局过量回忆杀；禁止无动作纯铺设定。

### 2. 短篇/短剧“5秒引爆”规则
- **视觉反差冲击**：第一眼必须极具反差感（如：首富穿破烂送外卖）。
- **情绪核弹点**：300字内必须完成一段落差极大的情绪拉扯（愤怒-震惊-懊悔）。
- **反常识金句钩子**：开篇第一句话必须打破常规逻辑。

## 三、 对话与风格（去AI味的 Prompt 技术）

### 1. AI味排除词典
- **禁用修辞**：“像...一样”、“仿佛...”、“由于...”等说明文式表达。
- **禁用高频降智词**：“一丝”、“直直的”、“闪过”、“坚定”、“不容置疑”、“在那一刻”。
- **句型改造**：
  - 用具体动词替代形容词（如“踉跄淌过泥水”替代“艰难地在雨中走”）。
  - 增加长短句错落感。
  - 对话占比必须达到30%以上，且对话需包含潜台词。

### 2. 逆向工程仿写（风格指纹提炼）
将经典作品（如猫腻、辰东的书）输入给AI，让AI反向总结其：
- **意象偏好**（喜欢用什么景物烘托）。
- **叙事节奏**（快慢交替规律）。
- **遣词造句习惯**。
随后固化为“风格指纹Prompt”，用于控制生成时的文风。

## 四、 题材打法矩阵

| 题材类型 | 核心驱动与风格 | AI执行重点提示 |
|:---|:---|:---|
| **玄幻/传统起点风** | 设定严谨、宏大世界觉醒、升级慢热 | 限制升级速度；强化环境描写；扩充等级设定库。 |
| **都市爽文/新媒体** | 快节奏打脸、情绪极度宣泄、钩子密 | 每500字要求一个期待点；强化“反派降智被狠狠打脸”的情绪描写。 |
| **言情/知乎短篇** | 第一人称强代入、多次反转虐恋 | 运用20条“虐”点公式；加强心理独白与信息错位陷阱。 |
| **宫斗/官场** | 权谋三十六计、智商在线的博弈 | 利用“信息差”、“借刀杀人”、“局中局”悬念公式计算剧情。 |
