# 语言治理规则

本文件专门处理长篇连载里“剧情没乱，但语言发硬、旁白失真、读者要自己翻译”的问题。

## 适用时机

遇到以下情况时，优先补读本文件而不是直接续写：

- 用户明确说“听不懂”“太抽象”“不像人话”
- 术语太多，读者第一次见就得停一下猜意思
- 军报、敌情、推理句像作者自己知道，但正文没说透
- 对白看着有劲，实际没答上问题
- 章里全是判断和气氛，缺动作、缺依据、缺后果

## 核心原则

1. 先保章节功能，再修语言。
2. 清晰可读，不等于统一口水化；但战争文、权谋文里的普通信息必须直给。
3. 判断句必须先落事实，再落人物判断，最后落到具体后果。
4. 能写清谁在做事、为什么做、结果变了什么，就不要拿术语、黑话、概念句顶上去。
5. 允许潜台词，不允许整段写完读者仍不知道谁要什么、谁怕什么、谁在拖时间。
6. 同一件事，不同人物不该共用一种说法；语言修正必须先看身份、性格和当下压力。
7. 不要写概括式、总结式、抽象提炼式句子；能讲场面，就别讲道理，能讲后果，就别讲大道理。
8. 称谓必须和人物身份、官职、场合一致；不能一句写成宦官，下一句又落成外朝官员。

## 硬闸门

### 1. 黑名单词句阻断

战争文、军报、敌情分析、权谋判断里，命中以下表达默认 `blocking=yes`，必须返工：

- `主口`
- `主杀招`
- `有鬼`
- `这一口` / `那一口`
- `压不住`
- `接刃`
- `错灯`
- `把眼睛拽过去`
- `事情严重了`
- `局势更坏了`
- `气息一下紧了`

说明：

- 不是这些词永远不能出现。
- 但只要它们承担的是关键信息，就必须在同句或下一句翻成人话。
- 如果删掉这些词，读者仍不知道敌人在打哪、主角在怕什么、慢一步会坏什么，这一句就不合格。

### 2. 首现术语必须就地解释

战争文第一次出现的术语，必须在同句或下一句补齐三件事：

1. 这是什么
2. 现在拿来做什么
3. 会影响谁

不合格示例：

- `回风高燧那边不能丢。`
- `白甲队先压前屏。`

合格方向：

- `回风高燧就是后头那座望风报讯的烽火台，那边一灭，狼山口就看不清外头还有没有后队。`
- `白甲队是专门顶最险那段墙的老卒队，先把他们压到前头，别让新兵先散。`

### 3. 战争文固定顺序

军报、敌情分析、主将判断、战场推理，默认必须按这个顺序写：

1. 先写现场事实
2. 再写人物判断
3. 最后写不处理会出的后果

不合格示例：

- `不是试探，是要继续压。`
- `东坡那边有鬼。`

合格方向：

- `后头还在上人，前排却只拿人往墙根垫，说明他们不是只想撞墙。再把人往主墙调，东坡那边就会顺着空档摸上来。`

### 4. 问答必须答题

角色 A 提问后，角色 B 的回答必须至少正面命中一项：

- 结论
- 原因
- 下一步动作

以下情况默认 `blocking=yes`：

- 上一句问“为什么这么判断”，下一句只抬情绪
- 上一句问“现在怎么办”，下一句只摆姿态
- 上一句问“是不是这一线更险”，下一句换个说法重复原句

### 5. 军令必须落成执行链

关键军令后，必须在 `1-3` 句内交代至少两项：

- 谁去做
- 先动哪批人
- 先动哪批物资
- 不这么做会出什么事

只有口号，没有执行动作，默认返工。

### 6. 作者概括腔阻断

以下写法默认高风险，命中就先删再重写：

- 作者替人物总结局势
- 用一句话概括一整段局面变化
- 用提炼式判断代替人物当场会说的话
- 用“本质上 / 说到底 / 真正的问题是 / 这意味着”这类总结腔替代场面推进
- 用抽象气氛词代替具体问题
- 用“看着有劲”的句子遮掉依据和后果
- 一整段都在解释“为什么重要”，却没有现场本身

### 7. 称谓错位阻断

以下情况默认高风险，命中就先返工：

- 前一句称对方 `公公`，后一句却落成 `郎中 / 侍郎 / 尚书 / 御史` 这类外朝官职
- 前一句叫法像主仆、内廷、军中，后一句身份却是完全另一套系统
- 称谓变化不是为了试探、讥讽、压人，而是作者自己叫错了

直接规则：

- 宦官才叫 `公公`
- 外朝文官优先按 `大人 / 官职`
- 不确定身份时，先写 `阁下 / 这位大人 / 怎么称呼`
- 若称谓一错会带偏人物关系、权力位置、场合气味，默认算硬错误，不算小瑕疵

## 概括句硬规则

### 1. 三类句子默认先删

- 概括局势：把整段场面压成一句结论
- 提炼判断：把人物判断写成作者提炼语
- 讲道理：不讲谁做了什么，只讲“问题在于”“本质上”

### 2. 三条改写方向

- 把“概括局势”的句子改成更具体的人话
- 把“提炼式判断”改成角色会说、读者能直接看见的表达
- 把“讲道理”改成“讲后果、讲场面、讲实际”

### 2.1 作者提炼句黑名单

以下句式若承担关键判断、局势总结、人物评价，默认按高风险处理：

- `真正的问题是`
- `关键在于`
- `本质上`
- `归根到底` / `归根结底`
- `说到底`
- `这意味着`
- `这说明`
- `说白了`
- `说穿了`
- `无非是`
- `像极了`
- `终究是`
- `从来都是`

处理原则：

- 不是这些词永远不能出现
- 但只要它们在替场面下结论、替人物提炼判断、替读者讲道理，就默认返工
- 优先改成：谁做了什么、哪条线坏了、慢一步会丢什么、这个人当场会怎么说

### 3. 直接对照

- 不写：`局势更坏了。`
  改写方向：`粮车今天进不了城，明天中营就得断火。`

- 不写：`这不是一场胜，这是把他们三年那口气都给掐断了。`
  改写方向：`这一仗打完，北朔那帮狗东西往后几年都别想在咱们面前直起腰。`

- 不写：`真正的问题是他们已经在逼我们退。`
  改写方向：`再把人往后撤，东坡那条路就要让给他们了。`

- 不写：`这意味着宗门已经到了生死关头。`
  改写方向：`护山阵快见底了，库房那几瓶丹也撑不了三天。`

- 不写：`这种催法，像极了当年京里最爱用的手段。`
  改写方向：`人刚进门就把日子卡死了，摆明了是不让他往后拖。`

## 高危信号

以下内容默认属于高风险：

- 句子看着有劲，读者却要自己翻译
- 术语字面解释了，但逻辑作用没解释
- 明明能直接说清，却反复绕成“那个人”“那件事”“某种真相”
- 连续多句都在讲局势，没有动作、资源、命令或结果变化
- 连续多句都在概括、提炼、下判断，却不让读者看见现场怎么坏下去
- 对白只有态度，没有交易、没有后果、没有推进
- 条件只写对我方成立，不写敌方为什么也能在同样天气、地形下行动
- 武将、文臣、主母、谋士、少年、老兵说话像同一个人
- 人物一到高压场景，全部自动切成同一种“利落成熟腔”
- 称谓和身份打架，读者刚判断出人物位置，下一句就被写乱

## 人物声口硬规则

### 1. 不要把“更清楚”修成“更一样”

- 清楚，是让读者听懂
- 不是把所有角色都修成同一套顺嘴话
- 修语言前先问：这句话更像谁会说的话，而不是“这样是不是更像作者想要的”

### 2. 先看角色，再决定怎么直白

- 武将 / 军中人：可以更直、更硬、更糙，优先落输赢、伤亡、脸面、守不守得住
- 文臣 / 幕僚：可以更稳、更收，优先落名分、责任、后果、规矩、牵连
- 老兵 / 底层人：优先落吃不吃得上、守不守得住、谁先死、哪儿先烂
- 世家 / 掌柜 / 内宅人物：各有自己的筹码语言，不能统一改成军中话或现代嘴炮

### 3. 压力上来时，允许声口变形

- 有些人急了会更短、更冲、更糙
- 有些人越急越收，话更少，但更扎人
- 有些人表面稳，动作先乱
- 所以不要只记“这个人平时怎么说”，还要记“他被逼急时怎么说”

### 4. 概括句改写时，优先按角色落地

- 不优先问：这句够不够高级
- 优先问：这句像不像这个人当场会骂出来、压出来、忍出来的话
- 若改写后虽然更清楚，但角色身份感被抹平，默认不算合格

## 默认修复顺序

1. 先删黑话和作者概括腔
2. 再把第一次出现的术语翻成人话
3. 再把概括句拆成场面、后果和执行动作
4. 再按角色身份、性格、压力重写声口
5. 再补“事实 -> 判断 -> 后果”
6. 再补问答对位和军令执行链
7. 最后才修节奏和文气

## 直接执行规则

- 能写“正面城墙”，就不要先写“主口”。
- 能写“真正要上的地方”，就不要先写“主杀招”。
- 能写“他们的人埋在东坡下头”，就不要只写“有鬼”。
- 能写“看错军情，把消息报错”，就不要只写“错灯”。
- 能写“守不住”，就不要只写“压不住”。
- 能写“真正打起来”，就不要只写“接刃”。
- 能写“把我们的兵力和注意都引到正面”，就不要写“把眼睛拽过去”。
- 能写“谁会死、哪会丢、东西会断在哪”，就不要只写“局势不好”“问题很大”。
- 能写“这件事会把哪条线压垮”，就不要写“真正的问题是”。
- 能写“人怎么说、怎么骂、怎么压人”，就不要写作者式提炼句。
- 能写“这位大人怎么称呼”，就不要在没确认身份时乱叫“公公”。

## 战争文最短自检

交稿前逐段问自己：

1. 普通读者能不能一遍看懂这句在说什么
2. 这里的术语是不是第一次出现，如果是，有没有就地解释
3. 这句判断的依据有没有写在前面
4. 这句回答有没有真的答上上一句
5. 这条命令有没有落到谁去做、先动什么、慢一步会怎样

任一项答不出，默认返工。

## 禁止误区

- 不要把“写得清楚”误判成“只能写成大白话”
- 不要把“更像散文”误判成“更像小说”
- 不要为了消 AI 味，把文本改得过度文学化
- 不要一边说“只改语言”，一边改掉情节结论
- 不要把作者脑内简称当成读者也会自动理解
