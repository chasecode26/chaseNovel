# 语言治理规则

本文件专门处理长篇连载里“剧情没乱，但语言发硬、旁白失真、读者要自己翻译”的问题。

## 适用时机

遇到以下情况时，优先补读本文件而不是直接续写：

- 用户明确说“听不懂”“太抽象”“不像人话”
- 术语太多，读者第一次见就得停一下猜意思
- 军报、敌情、推理句像作者自己知道，但正文没说透
- 对白看着有劲，实际没答上问题
- 章里全是判断和气氛，缺动作、缺依据、缺后果

## 核心原则

1. 先保章节功能，再修语言。
2. 清晰可读，不等于统一口水化；但战争文、权谋文里的普通信息必须直给。
3. 判断句必须先落事实，再落人物判断，最后落到具体后果。
4. 能写清谁在做事、为什么做、结果变了什么，就不要拿术语、黑话、概念句顶上去。
5. 允许潜台词，不允许整段写完读者仍不知道谁要什么、谁怕什么、谁在拖时间。

## 硬闸门

### 1. 黑名单词句阻断

战争文、军报、敌情分析、权谋判断里，命中以下表达默认 `blocking=yes`，必须返工：

- `主口`
- `主杀招`
- `有鬼`
- `这一口` / `那一口`
- `压不住`
- `接刃`
- `错灯`
- `把眼睛拽过去`
- `事情严重了`
- `局势更坏了`
- `气息一下紧了`

说明：

- 不是这些词永远不能出现。
- 但只要它们承担的是关键信息，就必须在同句或下一句翻成人话。
- 如果删掉这些词，读者仍不知道敌人在打哪、主角在怕什么、慢一步会坏什么，这一句就不合格。

### 2. 首现术语必须就地解释

战争文第一次出现的术语，必须在同句或下一句补齐三件事：

1. 这是什么
2. 现在拿来做什么
3. 会影响谁

不合格示例：

- `回风高燧那边不能丢。`
- `白甲队先压前屏。`

合格方向：

- `回风高燧就是后头那座望风报讯的烽火台，那边一灭，狼山口就看不清外头还有没有后队。`
- `白甲队是专门顶最险那段墙的老卒队，先把他们压到前头，别让新兵先散。`

### 3. 战争文固定顺序

军报、敌情分析、主将判断、战场推理，默认必须按这个顺序写：

1. 先写现场事实
2. 再写人物判断
3. 最后写不处理会出的后果

不合格示例：

- `不是试探，是要继续压。`
- `东坡那边有鬼。`

合格方向：

- `后头还在上人，前排却只拿人往墙根垫，说明他们不是只想撞墙。再把人往主墙调，东坡那边就会顺着空档摸上来。`

### 4. 问答必须答题

角色 A 提问后，角色 B 的回答必须至少正面命中一项：

- 结论
- 原因
- 下一步动作

以下情况默认 `blocking=yes`：

- 上一句问“为什么这么判断”，下一句只抬情绪
- 上一句问“现在怎么办”，下一句只摆姿态
- 上一句问“是不是这一线更险”，下一句换个说法重复原句

### 5. 军令必须落成执行链

关键军令后，必须在 `1-3` 句内交代至少两项：

- 谁去做
- 先动哪批人
- 先动哪批物资
- 不这么做会出什么事

只有口号，没有执行动作，默认返工。

### 6. 作者概括腔阻断

以下写法默认高风险，命中就先删再重写：

- 作者替人物总结局势
- 用抽象气氛词代替具体问题
- 用“看着有劲”的句子遮掉依据和后果
- 一整段都在解释“为什么重要”，却没有现场本身

## 高危信号

以下内容默认属于高风险：

- 句子看着有劲，读者却要自己翻译
- 术语字面解释了，但逻辑作用没解释
- 明明能直接说清，却反复绕成“那个人”“那件事”“某种真相”
- 连续多句都在讲局势，没有动作、资源、命令或结果变化
- 对白只有态度，没有交易、没有后果、没有推进
- 条件只写对我方成立，不写敌方为什么也能在同样天气、地形下行动

## 默认修复顺序

1. 先删黑话和作者概括腔
2. 再把第一次出现的术语翻成人话
3. 再补“事实 -> 判断 -> 后果”
4. 再补问答对位和军令执行链
5. 最后才修节奏和文气

## 直接执行规则

- 能写“正面城墙”，就不要先写“主口”。
- 能写“真正要上的地方”，就不要先写“主杀招”。
- 能写“他们的人埋在东坡下头”，就不要只写“有鬼”。
- 能写“看错军情，把消息报错”，就不要只写“错灯”。
- 能写“守不住”，就不要只写“压不住”。
- 能写“真正打起来”，就不要只写“接刃”。
- 能写“把我们的兵力和注意都引到正面”，就不要写“把眼睛拽过去”。

## 战争文最短自检

交稿前逐段问自己：

1. 普通读者能不能一遍看懂这句在说什么
2. 这里的术语是不是第一次出现，如果是，有没有就地解释
3. 这句判断的依据有没有写在前面
4. 这句回答有没有真的答上上一句
5. 这条命令有没有落到谁去做、先动什么、慢一步会怎样

任一项答不出，默认返工。

## 禁止误区

- 不要把“写得清楚”误判成“只能写成大白话”
- 不要把“更像散文”误判成“更像小说”
- 不要为了消 AI 味，把文本改得过度文学化
- 不要一边说“只改语言”，一边改掉情节结论
- 不要把作者脑内简称当成读者也会自动理解
