# 主角人设与情感套路技法库 (Tropes & Archetypes Library)
此文件作为 `chaseNovel` 中写人设、写情感线、写虐点与爽点时的强制检阅素材。

## 一、 20条经典虐点/情感起伏曲线 (用于言情/女频/跌宕起伏片段)
当写作目标为“制造感情冲突”、“催泪”、“虐心”时，必须调用以下公式之一，并在 `state.md` 或章卡中明示。

1. **替身暴击公式**
   - **要素**：旧物暴露/无意间的称呼错乱 + 情感对比。
   - **曲线**：女配主动挑衅/男主无意流露 → 女主震惊心跳加速 → 发现真相（自己只是替身） → 从不信到绝望 → 撕裂醒悟。
2. **死别公式**
   - **要素**：临死前的微小承诺无法兑现 + 活人的后知后觉。
   - **重点**：死亡不虐，虐的是“原定明天要一起去看的海，但今天人没了”。
3. **白月光回归公式**
   - **要素**：两人刚建立起信任或温情时刻 + 白月光一个电话/出现导致男主立刻抛下女主。
4. **追妻火葬场公式**
   - **前期**：男主视女主如草芥，随意践踏真心。
   - **节点**：女主心灰意冷彻底离开（假死/失忆/决绝转投他人）。
   - **后期**：男主发现原来自己爱她，开始卑微挽回但遭受冷眼。

（其余虐点依此类推：误会公式、权力与感情的二选一、失忆背叛等。全流程必须经历：期盼 → 发现异常 → 证实绝望 → 做出决断）。

## 二、 72爆款主角设定 & 50性格矩阵

### 1. 热门“钩子”人设 (自带冲突属性)
在 `/一键开书` 时，应该优先挑选自带张力的人设：
- **疯批权臣/反派**：高智商，没有道德底线，但只对一人心软（极致反差）。
- **落魄/觉醒替身**：从前隐忍卑微，一朝觉醒直接断绝关系并搞事业。
- **满级大佬重回新手村**：用绝对实力碾压，强行降维打击制造极速爽感。
- **系统叛变者**：不想做既定任务偏要逆天改命的人。
- **时间囚徒/重生者**：拥有未来记忆，利用信息差步步为营。

### 2. 性格矩阵配置规则
主角性格绝不能是单一的“善良”。必须配置为：
- **主性格 (核心颜色)**：如冷静、坚毅、腹黑。
- **反差性格 (触发条件)**：如“极度护短，触逆鳞者死”或“平时懒散，拔剑必杀”。
- **成长缺陷 (驱动弧线)**：如缺乏信任感、有心结未解。

## 三、 爽点套路 (悬念与收割库)

- **扮猪吃虎**：压抑（被鄙夷/打压）时长需满足3章以内，必须配以一个不可反驳的打脸。
- **借刀杀人**：主角不亲自下场，通过语言/信息差引导反派自相残杀。
- **降维打击**：利用现代知识碾压古代、利用高阶功法碾压低阶，制造错位爽感。
- **局中局/计中计**：读者以为主角中计了，结果这是主角故意布下的更大的局。
