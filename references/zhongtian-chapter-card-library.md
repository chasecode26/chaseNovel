# 种田文章节卡样例库

---

## 1. 第一轮收成章

```markdown
# 第 9 章写作方案
- 本章功能：兑现积累感
- 本章冲突：天气、劳力、技术细节或旁人不看好
- 本章结果类型：资源到手型
- 章尾钩子类型：资源争夺型
- 本章章尾钩子：收成刚见起色，就有人想低价吃掉主角的成果
```

## 2. 合作试探章

```markdown
# 第 18 章写作方案
- 本章功能：推进关系网络
- 本章冲突：邻里、家族或合作方不完全信任主角
- 本章结果类型：小胜型
- 章尾钩子类型：关系突变型
- 本章章尾钩子：原本观望的人决定站过来，但会带来新的利益问题
```

## 3. 产业升级章

```markdown
# 第 31 章写作方案
- 本章功能：从单点种植升级成经营闭环
- 本章冲突：技术、资金、人手或渠道短缺
- 本章结果类型：赢局面丢关系型
- 章尾钩子类型：结果未揭晓型
- 本章章尾钩子：新产业刚起步，外部价格或官面规则突然变化
```

## 4. 天灾应对章

```markdown
# 第 44 章写作方案
- 本章功能：打破稳态
- 本章冲突：旱灾、水灾、虫灾、市场崩盘之一
- 本章结果类型：保命失机型
- 章尾钩子类型：危机压顶型
- 本章章尾钩子：主角保住了核心资源，但更大的连续风险才刚开始
```

---

一句话原则：

**种田文章卡不能只写“今天又多赚了一点”，要写“这次积累改变了谁、引来了什么、下一步怎样升级”。**
