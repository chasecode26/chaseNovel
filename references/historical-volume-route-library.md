这份文档解决的是整卷推进，而不是单章处理。

**让历史权谋 / 边境战争一卷一卷往前长，不靠重复军议、重复打仗、重复朝堂吵架。**

## 一、卷级路线的基本原则

历史权谋 / 边境战争每一卷都至少要换掉一件事：

1. 主要战场
2. 主要权力关系
3. 主角的身份位置
4. 本卷核心代价

如果只是换地图、换敌将名字、换一批臣子，但主角还是用同一套方式赢，同样会疲劳。

## 二、五条主卷路线

### 路线 1：边关立足卷

适合：

- 开书前期
- 主角还没真正拿到军权

本卷核心目标：

- 在边地站住
- 让军中、地方、上位者第一次承认主角不可忽视

本卷核心矛盾：

- 主角想活，别人只想用他
- 边军想守，朝堂想控

本卷四个关键节点：

1. 初到边地，身份轻、处境险
2. 靠一次判断或一次小战果立住基本盘
3. 军中或地方体系第一次公开压主角
4. 主角拿到实权雏形，但也被正式盯上

本卷结尾最适合留下：

- 军权到手
- 圣意未明
- 有人开始怕主角成势

### 路线 2：军权争夺卷

适合：

- 主角已在边地有威望
- 需要把“能打”升级成“能统”

本卷核心目标：

- 拿到更稳定的调兵权、粮权、处分权

本卷核心矛盾：

- 老将、监军、地方官、朝廷使者都不想让主角真正做主

本卷四个关键节点：

1. 主角明面被抬，暗里被卡
2. 出现一次“违令有功”或“守规会死”的困局
3. 主角必须选制度还是人情
4. 主角坐稳位置，但代价是关系裂痕

本卷结尾最适合留下：

- 军权稳了
- 旧关系坏了
- 新敌人不在军中，在朝中

### 路线 3：朝堂咬合卷

适合：

- 主角已经不是纯边将
- 需要把线从地方拉回朝堂

本卷核心目标：

- 让主角第一次真正进入高层权力结构

本卷核心矛盾：

- 打赢不代表能活
- 上位者既要你有用，又不能让你太自由

本卷四个关键节点：

1. 主角因战功或名望被推入朝堂视野
2. 封赏与制衡一起落下
3. 原盟友、门阀、台谏各自表态
4. 主角被迫公开站队或公开拒绝站队

本卷结尾最适合留下：

- 拿到新名分
- 丢掉中立空间
- 更大权斗开门

### 路线 4：外战转内斗卷

适合：

- 读者已经看多了正面战场
- 需要换打法

本卷核心目标：

- 把敌人从城外移到城内、营内、朝内

本卷核心矛盾：

- 真正最危险的，不再是敌军，而是自己体系里的漏洞和背刺

本卷四个关键节点：

1. 表面外战顺利
2. 后勤、粮道、军令、情报链出现内鬼问题
3. 主角发现旧盟友或自己人也在借机布局
4. 主角赢下外战，却引爆内斗

本卷结尾最适合留下：

- 外敌暂压
- 内部裂开
- 下一卷从“打别人”转成“整自己体系”

### 路线 5：名望反噬卷

适合：

- 主角已成势
- 需要防止一路无脑顺推

本卷核心目标：

- 让读者看到“主角越强，越难安全”

本卷核心矛盾：

- 以前想出头，现在出头成了负担

本卷四个关键节点：

1. 主角威望达到新高度
2. 各方开始借主角名望做文章
3. 主角发现自己被推到不能退的位置
4. 主角必须主动砍掉一部分名望收益来保后手

本卷结尾最适合留下：

- 主角更强了
- 但也正式成为必须被先手处理的人

## 三、卷与卷之间怎么接

推荐的常用接法：

- `边关立足卷 -> 军权争夺卷`
- `军权争夺卷 -> 朝堂咬合卷`
- `朝堂咬合卷 -> 外战转内斗卷`
- `外战转内斗卷 -> 名望反噬卷`

也可以反接：

- 如果前一卷压朝堂过重，下一卷就回边关
- 如果前一卷全是外战，下一卷就优先内斗

一句话原则：

**下一卷要接上一卷的后账，而不是重新开一套。**

## 四、每卷最该换什么

### 卷 1 到卷 2

- 换“主角位置”
- 从求活，换到争权

### 卷 2 到卷 3

- 换“敌人层级”
- 从地方，换到朝堂

### 卷 3 到卷 4

- 换“冲突方向”
- 从外战，换到内斗

### 卷 4 到卷 5

- 换“主角代价”
- 从求赢，换到求稳、求活、求留后手

## 五、卷末收束的正确方式

历史权谋 / 边境战争的卷末，不适合收成纯胜利庆功。

更好的卷末结构是：

1. 本卷核心目标兑现一半到八成
2. 本卷主要关系至少裂一条或换一条
3. 本卷主角位置明显改变
4. 下一卷入口清晰，但不是另起炉灶

常见卷末留口：

- 新军权到手，但旧人心不稳
- 战场打赢了，但朝堂开始下手
- 敌军压住了，盟友开始变质
- 主角升上来了，却失去了原来的退路

## 六、快检清单

做一卷前，先问：

1. 这卷比上一卷主要换了什么？
2. 这卷的最大敌人是在外面、里面，还是上面？
3. 这卷结尾主角的位置会发生什么变化？
4. 这卷最重要的代价是什么？
5. 这卷结束后，下一卷是接后账还是重开套路？

如果 1-5 说不清，这卷大概率会散。
