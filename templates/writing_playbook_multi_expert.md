# AI多专家协作工作流 (Multi-Expert Prompt Setup)
这是基于飞书顶尖AI写作攻略提炼的「高阶结构化出稿模式」。
在起草单章前，如果你需要写极其复杂的权谋、高燃的群战或深刻的感情戏，请调用此工作流，让AI扮演一组专家委员会进行联合生成。

## 工作流启动提示词 (启动模板)
```markdown
请你调用内部的【专家委员会模式】协助我创作这一章。
在最终输出正文前，必须依次由以下五位大师进行一轮打磨与思考，你要展示他们思考的过程，然后综合结论输出最终完美无瑕的一章：

1. 【情节大师】：负责审视当前大纲，保证这章不要平铺直叙。必须设计3个“冲突递进点”或1个“意料之外的波折”。
2. 【渲染大师】：负责对战斗、打脸、情感爆发进行极致描绘。禁止用“很强”，必须通过侧面环境破灭、路人惊恐的侧写反馈来烘托。确保加入听觉和触觉。
3. 【逻辑大师】：负责检查“降智”漏洞和幻觉。确认反派行为符合其社会地位！确认主角使用的底牌是前文伏笔或合理的。
4. 【对白大师】：负责砍掉30%的冗余对白。所有台词必须带着角色的潜台词，甚至阴阳怪气。让语言锋利如刀。
5. 【悬念主编】：最后拍板，强行从本章的最高潮或危机点“一刀切断”，写出吊足胃口的章尾钩子并直接转入下章预告感。

请开始你们的专家讨论，并最终输出符合上述所有要求、去AI味、爽点密集的2500字章节。
```
