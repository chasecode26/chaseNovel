# 去AI味润色与风格校准流水线 (Vibe Check Workflow)
此模板用于章节初稿完成后的“风格校准”与“润色”阶段。

## 1. 核心目标：五层校准审计
执行 `/润色` 或写入正稿前，强制经过这五层重塑：

**第一层：AI特征词黑名单（拦截级）**
一旦出现以下词汇，立刻触发警告，必须重写该句：
- `像...一样` / `仿佛...` (过度比喻)
- `由于...` / `导致了...` / `事实上...` (论文式说明)
- `那一刻` / `就在这时` / `此时此刻` (生硬转折)
- `不禁...` / `不由得...` / `闪过一丝...` (套路化心理描写)
- `值得注意的是` / `不得不说` / `可以说` (第三视角上帝解说)
- `不容置疑的语气` / `坚定地回答` / `眼神变得深邃` (标签化表情包)

**第二层：句式节奏审计**
- 检查连续长句/短句段落。打斗、生死危机必须全都切碎为6-10字的短句。
- “刀光闪。血珠溅。他倒退两步，咽下喉咙里涌上的腥甜。”

**第三层：感官密度检查**
- 每500字至少有1处视觉+1处非视觉（听觉/嗅觉/触觉）描写。
- 去除抽象形容词（如“艰难”），全用动词驱动（如“趟过泥水，身子踉跄”）。

**第四层：对话质量与潜台词**
- 不允许直接把心里话说出来。
- 将无效解释改写为姿态：“我很生气” -> “把桌上的茶杯扫落在地，手背青筋暴起，但他只吐出一个字：好。”

**第五层：风格指纹注入 (Style Fingerprint)**
- 如果在 `state.md` 或本章卡中指定了具体作家的风格（如：猫腻风、辰东风、唐家三少风），必须加载 `templates/style_fingerprints.md` 中对应的指纹卡，严格按照其意象偏好和对话密度进行最终重构。

## 2. 润色Prompt 示例 (供调用)
```markdown
请你进行深度润色重构。
要求如下：
1. 【过滤】完全消除AI味，禁用黑名单词汇（如“然而”、“就在这时”、“不禁”）。
2. 【感官】将所有抽象形容词翻译为具体的微动作和五感细节。
3. 【节奏】根据当前的紧张程度调整句段长度，使用动词驱动推进画面。
4. 【潜台词】对话必须展现人物潜台词，不要干巴巴的宣告。
5. 【指纹】本次润色请加载 `style_fingerprints.md` 中的【XXXX风】，在遣词造句和意象选择上向其无缝靠拢。
```
