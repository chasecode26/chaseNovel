# 开篇路线速查卡

用途：
- `/一键开书` 前先选第一章路线
- 第一章总不抓人时快速换路

原则：
- 先选 `1` 条主路线，再写第一章
- 路线服务题材、卖点和当前压力源

## 三步判断

1. 第一章最该立什么：危机 / 卖点 / 气质 / 处境 / 生存压力
2. 读者为什么现在继续看：要死要输 / 会翻身 / 背后有局 / 世界失控 / 关系卡死
3. 这一章最怕什么：太慢 / 太空 / 太像背景介绍 / 太有气质但不推进

## 自动推荐

- 强异常、悬疑、诡异：`危机直砸型`
- 底层成长、凡人流、种田成长：`普通人入局型`
- 历史、权谋、群像、厚重长篇：`气质压场型`
- 热血玄幻、男频逆袭：`热血反压型`
- 朝堂权谋、多层身份、智斗古代文：`权谋文气型`
- 末世囤货、重生生存：`重生囤货型`
- 系统流末世、高反馈生存：`系统囤货型`
- 苟道修仙、长生慢发育：`长生慢发育型`
- 强调避险判断：`趋吉避凶型`
- 强调藏锋潜伏：`伪装潜伏型`

冲突时按：`题材 > 卖点 > 当前压力源`

## 路线卡

### 危机直砸型
- 先立：眼前异常、当下危险、主角第一反应
- 适合：悬疑、诡异、强异常设定
- 禁忌：先铺世界观、先讲背景

### 普通人入局型
- 先立：主角是谁、缺什么、改命入口
- 适合：凡人流、底层成长、种田成长
- 禁忌：太多日常没钩子，主角完全被动

### 气质压场型
- 先立：叙事气味、人物姿态、世界压力
- 适合：历史、权谋、仙侠群像
- 禁忌：只有格调，没有事件

### 热血反压型
- 先立：受压、尊严受损、翻身承诺
- 适合：热血玄幻、逆袭爽文
- 禁忌：只有委屈，没有压迫机制和兑现轮廓

### 权谋文气型
- 先立：人物不简单、背后有局、表面之外还有一层
- 适合：朝堂权谋、多层身份
- 禁忌：只靠神秘，不给现实抓手

### 重生囤货型
- 先立：上一世损失、为什么现在必须囤、囤货行动立刻启动
- 适合：末世囤货、重生生存
- 禁忌：列清单，没有抢时间压力

### 系统囤货型
- 先立：系统可用、收益可见、现实局势会被改写
- 适合：系统流末世、高反馈生存
- 禁忌：系统播报流水账，只爽不进局

### 长生慢发育型
- 先立：为什么必须稳、稳的收益是什么
- 适合：苟道修仙、长生发育
- 禁忌：只讲道理，不给实际动作

### 趋吉避凶型
- 先立：死局感、避险能力、第一次判断
- 适合：风险判断型苟道
- 禁忌：只说能避险，不给实证

### 伪装潜伏型
- 先立：为什么暴露即死、主角怎么藏
- 适合：潜伏、卧底、伪装成长
- 禁忌：只有低调口号，没有具体藏法
