# 角色说话差分表

用途：

- 给 `StyleDialogue` 一个最短可执行的角色说话区分骨架
- 避免人物都用同一种句长、同一种判断语、同一种情绪爆发方式

```markdown
# 角色说话差分表

## 角色 1
- 身份 / 关系位置：
- 平时句长：短 / 中 / 长
- 说话速度：快 / 稳 / 慢
- 常用措辞：
- 常用动作伴随：
- 情绪上来时会怎么变：
- 他绝不会说的话：
- 他最像哪种气味：冷 / 刚 / 软 / 绕 / 稳 / 轻 / 脏

## 角色 2
- 身份 / 关系位置：
- 平时句长：
- 说话速度：
- 常用措辞：
- 常用动作伴随：
- 情绪上来时会怎么变：
- 他绝不会说的话：
- 他最像哪种气味：

## 对话撞车快检
- 两个角色会不会都用同类判断句：
- 两个角色会不会都爱解释：
- 两个角色在压力下是不是还说得一样整齐：
- 若撞车，优先拉开什么：
```

使用规则：

- 先拉开句长、语速、措辞，再谈“文艺感”
- 角色区分优先靠身份、欲望、压力，不靠奇怪口头禅
- 如果一个角色删掉名字还能认出来，差分才算成立
