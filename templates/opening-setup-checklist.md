# 开书骨架补全清单

> 这份清单不是让你把世界观写成百科，而是防止“能起稿、不能长写”。
> `/一键开书`、题材规划、卷纲初始化时，先按题材补齐对应骨架。
> 原则：只记录后续会反复用到、缺了就容易写崩的结构信息。

## 通用硬规则

无论什么题材，开书时至少先补齐这 6 类：

- `主角起手盘`：现在有什么、缺什么、最怕失去什么
- `核心冲突盘`：眼前问题、阶段目标、长期要争的东西
- `资源盘`：钱、权、人、地、物、信息里哪几项最关键
- `信息盘`：谁知道真相、谁知道一半、谁被误导
- `升级盘`：这书后面靠什么持续推进，而不是只靠开局异常
- `反派池`：至少准备 `3-5` 个不同路数的压力源，不要只有一个脸谱反派

如果这 6 类里有两类以上答不上来，默认说明“能开头，不能稳定连载”。

## 历史 / 权谋

开书前至少补齐：

- `权力层级图`：皇帝 / 中枢 / 宗室 / 外戚 / 地方 / 军府 / 监察分别谁说了算
- `朝堂议题池`：本书前期真正会反复争的事是什么
  例如：边军、粮税、军功、封赏、储位、和战、盐铁、赈灾
- `兵力与地理盘`：谁手里有多少兵、卡在哪些关口、哪几处地形最要命
- `情报系统`：密探、驿站、亲兵、商路、内廷、边军各自怎么传消息，谁能截断
- `财政与粮道盘`：钱粮从哪来，军需怎么走，哪一环最容易被人做手脚
- `多方势力表`：本国内部、边军系统、地方豪强、外敌、盟友分别想要什么
- `朝堂体系`：官职升降、军政关系、监军/文官/武将怎么互相制衡
- `反派池`：至少准备 `朝堂反派 + 地方反派 + 军中反派 + 外部敌手` 四类

不补这些，后面最容易出现：

- 权谋只靠嘴遁
- 战争只剩战报
- 朝堂像吵架群
- 反派只有一个人在跟主角作对

## 玄幻 / 修仙

开书前至少补齐：

- `力量体系`：前中期会用到的境界、代价、突破门槛，不要只写名字
- `资源体系`：灵石、丹药、传承、法器、洞府、机缘分别怎么获取
- `宗门 / 地域图`：主角当前在哪，下一阶段会去哪里，跨地图为什么成立
- `势力关系盘`：宗门、世家、散修、皇朝、异族之间怎么咬合
- `禁忌规则`：什么事一旦暴露会立刻出大问题
- `反派池`：同辈竞争者、老一辈压制者、外部势力掠夺者、隐性大敌

## 仙侠 / 苟道

开书前至少补齐：

- `暴露代价`：主角为什么必须苟，一旦暴露会死在哪
- `隐藏边界`：哪些能力能藏，哪些资源不能露
- `安全区与危险区`：哪里能苟，哪里不能久留
- `资源慢循环`：主角靠什么稳定变强，不靠天降大礼包
- `风险升级链`：随着主角变强，会引来哪几层新风险
- `反派池`：同门窥伺、宗门规训、外部猎杀、旧仇追索

## 都市 / 系统流

开书前至少补齐：

- `系统落地规则`：奖励怎么转成现实变化，代价和限制是什么
- `现实赛道`：主角混的是哪条线
  例如：职场、直播、校园、创业、文娱、资本、豪门、神豪
- `社会站位盘`：主角现在处在什么阶层，往上要跨哪几层门槛
- `现实关系网`：家人、同事、上级、同行、资本、客户分别怎么卡位
- `舆论 / 信息盘`：消息怎么扩散，谁能带节奏，谁能压消息
- `升级闭环`：前 30 章主要靠赚钱、名气、资源、人脉、地位还是规则反制推进
- `反派池`：至少准备 `眼前反派 + 制度型反派 + 更高层对手`

不补这些，后面最容易出现：

- 系统只会播报
- 反派只会狗眼看人低
- 都市线没有行业和现实重量

## 种田 / 基建

开书前至少补齐：

- `家庭盘`：这家里谁能干活、谁拖后腿、谁最可能闹事
- `土地与生产盘`：地、水、种子、农具、牲口、手艺最缺什么
- `季节盘`：当前是什么时令，未来几轮会受什么季节约束
- `债务与生计盘`：现在最紧的是粮、钱、税、药还是劳力
- `村庄 / 乡绅 / 族老结构`：谁能卡主角，谁能帮主角，谁会摇摆
- `交易链`：东西产出来卖给谁，怎么运，谁抽成
- `产业升级链`：从种粮到养殖、作坊、商路，后面靠什么升级
- `反派池`：族内麻烦、乡里压迫、市场掐价、官差 / 地痞 / 天灾

## 末世 / 废土生存

开书前至少补齐：

- `威胁生态`：感染者 / 变异体 / 掠夺者 / 极端天气 / 辐射分别怎么杀人
- `地图盘`：据点、安全区、危险区、补给点、撤离线怎么分布
- `物资盘`：水、粮、药、武器、电、交通里哪项最缺
- `团队角色盘`：谁负责战斗、后勤、医疗、判断、守点
- `秩序盘`：据点内部按什么规则活，违反规则会怎样
- `情报盘`：无线电、拾荒者、外出队、旧网络、幸存者口口相传谁更可信
- `倒计时`：尸潮、缺药、断电、暴雨、冬季、据点暴露里至少要有一个真实时限
- `反派池`：外部掠夺者、内部野心者、安全区上位者、非人威胁

## 末世 / 囤货

开书前至少补齐：

- `囤货目标表`：到底囤什么，优先级怎么排
- `空间 / 仓储边界`：能装多少，怎么转运，怎么掩护
- `灾变时间线`：什么时候开始乱，什么时候全面失控
- `前期社会秩序`：灾变前怎么买、怎么运、怎么避开注意
- `中后期消耗线`：囤货不是清单，而是消耗、补充、交换和防守
- `反派池`：亲友消耗、邻里觊觎、组织掠夺、规则崩盘后的武力冲突

## 盗墓 / 民国奇诡

开书前至少补齐：

- `墓外线`：为什么下墓，谁出钱，谁压着主角去
- `墓内规则`：机关、风水、异象、禁忌哪些是真会反复用到的
- `文物流转链`：摸出来以后卖给谁、藏到哪、谁会盯上
- `势力盘`：军阀、帮会、同行、学界、洋人、黑市怎么咬合
- `历史线索盘`：墓和哪段旧事挂钩，旧案会怎么反咬现实
- `队伍结构`：谁是掌眼、谁是下手、谁最可能反水
- `反派池`：同行、雇主、旧仇、墓里东西本身带出的后患

## 怎么用

开书时建议按这个顺序：

1. 先定题材和主玩法
2. 再补对应题材骨架
3. 再写一句话卖点
4. 再做黄金三章
5. 最后才进正文

不要反过来。

先写了三章，再回头补这些，通常已经开始埋错雷了。
