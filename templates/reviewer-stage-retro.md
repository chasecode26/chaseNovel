# Reviewer 阶段复盘模板

用途：

- 给 `Reviewer` 一个短而硬的阶段复盘回写骨架
- 复盘结果直接服务下一阶段，不扩成大段分析

```markdown
# 阶段复盘

## 当前疲劳点
- 哪几章开始发水 / 发平：
- 疲劳主要来自什么：

## 当前重复点
- 重复的是桥段 / 结果 / 钩子 / 人物组合中的哪一类：
- 近几章最像的地方：

## 当前拖欠承诺
- 哪个承诺拖太久：
- 若继续拖，会先伤哪类读感：

## 下一阶段纠偏动作
- 下一章必须补什么：
- 接下来 3-5 章必须换什么：
- 哪条线暂时不能再拖：
```

使用规则：

- 一次复盘只写最要命的 4 项，不做长篇总述
- 复盘结论优先写回 `state.md`、`findings.md` 或阶段摘要
- 如果复盘后仍说不清下一章怎么纠偏，默认返回 `Planner`
