# 章尾钩子速查卡

用途：
- `/写` 前先选章尾钩子路线
- 收尾发平时快速补强追读动力

原则：
- 先看本章功能，再选钩子
- 一章只保 `1` 个主钩子，最多叠 `1` 个副压力
- 钩子必须改变下一章行动

## 三步判断

1. 下一章最该立刻处理什么：结果揭晓 / 风险落下 / 选择到来 / 真相翻盘 / 关系变化 / 资源争夺 / 欲望升级
2. 读者此刻最关心什么：输赢 / 暴露 / 取舍 / 真相 / 关系 / 机会 / 更大目标
3. 本章最怕什么：赢完就散 / 危机停在嘴上 / 情绪有了但没事件 / 只有悬念没有代价 / 和前几章太像

## 自动推荐

- 本章刚兑现爽点：`危机压顶型` 或 `资源争夺型`
- 本章推进主线、即将进入下一步行动：`结果未揭晓型` 或 `选择逼近型`
- 本章埋设伏笔、刷新认知：`信息反转型`
- 本章推进关系或站队变化：`关系突变型`
- 本章完成小目标要抬更大目标：`欲望升级型`

冲突时按：`本章功能 > 当前压力源 > 题材惯性`

## 路线卡

### 结果未揭晓型
- 适合：比试、考核、谈判、奖励结算
- 先立：结果马上揭晓，且会改变主角处境
- 禁忌：硬停在“然后他看了过去”

### 危机压顶型
- 适合：主角刚赢一手、身份可能暴露、强敌到门口
- 先立：具体危险已经落地，下一章必须立刻应对
- 禁忌：只写“危险正在逼近”

### 选择逼近型
- 适合：站队、取舍、道义与利益冲突
- 先立：主角必须选，且每个选项都有代价
- 禁忌：选项其实只有一个

### 信息反转型
- 适合：悬疑、权谋、旧结论被推翻
- 先立：旧认知被击穿，新信息会重排后续行动
- 禁忌：只搞神秘，不给反转锚点

### 关系突变型
- 适合：暧昧推进、联盟破裂、公开站队、身份掉马
- 先立：关系已经变了，而且会逼出下一章动作
- 禁忌：只有心动，没有事件

### 资源争夺型
- 适合：机缘、名额、物资、传承、职位、订单
- 先立：好东西出现，不抢就会失去
- 禁忌：只有资源名词，没有争夺关系和时限

### 欲望升级型
- 适合：小目标完成后抬长期野心
- 先立：主角接下来想争更大的东西，代价也更高
- 禁忌：只是嘴上喊口号，没有新路线

## 快检

- 钩子有没有具体问题、具体风险或具体代价
- 钩子是不是和当前主线相关
- 钩子是否和前几章同质
- 下一章第一段是否已经被这个钩子决定
