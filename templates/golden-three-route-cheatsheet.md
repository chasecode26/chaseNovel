# 黄金三章路线速查卡

用途：

- `/一键开书` 时，把开篇主路线直接扩成前 3 章功能链
- 第一章能抓人，但第二三章容易发散时，快速校正
- 想判断前三章是不是只在重复卖点，而没有层层抬高时，直接对照

原则：

- 第一章负责抓入场
- 第二章负责让主角开始行动
- 第三章负责给长期承诺并挂强钩子
- 三章要递进，不要只是同一种爽点换场景重放

---

## 用法

先回答 3 个问题：

1. 第一章已经立住了什么？
   - 危机
   - 处境
   - 气质
   - 屈辱
   - 大局
   - 生存窗口
   - 苟道规则

2. 第二章最该补什么？
   - 主角主动性
   - 新阻力
   - 新规则
   - 新机会
   - 新关系
   - 新代价

3. 第三章必须让读者看到什么？
   - 这书后面怎么爽
   - 这书后面怎么斗
   - 这书后面怎么活
   - 这书后面怎么苟
   - 这书后面怎么爱恨翻涌

答完后，按下面的路线表落地。

---

## 自动推荐规则

当用户没有明确指定黄金三章节拍时，默认按以下顺序自动推荐：

1. 先看开篇主路线
   - `危机直砸型`：三章链优先 `异常 -> 应对 -> 更大异常`
   - `普通人入局型`：三章链优先 `处境 -> 入局 -> 路线承诺`
   - `气质压场型`：三章链优先 `气质与处境 -> 局面显形 -> 阵营与长期局`
   - `热血反压型`：三章链优先 `受压 -> 第一次反手 -> 升级线承诺`
   - `权谋文气型`：三章链优先 `人不简单 -> 局不简单 -> 站队与暗线`
   - `重生囤货型`：三章链优先 `末世将至 -> 抢时间囤货 -> 生存线与敌我线`
   - `系统囤货型`：三章链优先 `系统可用 -> 囤货见收益 -> 更大生存目标`
   - `长生慢发育型`：三章链优先 `必须稳 -> 稳住第一步 -> 长期发育承诺`
   - `趋吉避凶型`：三章链优先 `死局感 -> 避险验证 -> 风险管理主线`
   - `伪装潜伏型`：三章链优先 `不能暴露 -> 先稳身份 -> 潜伏成长路线`

2. 再看核心卖点
   - 卖点是异常与真相：向“第一章异常、第二章追索、第三章翻大问题”偏
   - 卖点是翻身与升级：向“第一章受压、第二章小反打、第三章长线承诺”偏
   - 卖点是囤货与求生：向“第一章抢时间、第二章证收益、第三章立敌我局”偏
   - 卖点是权谋与布局：向“第一章立视角、第二章显局、第三章抬站队成本”偏
   - 卖点是苟道与发育：向“第一章立风险、第二章证稳法、第三章给长期安全边际”偏

3. 最后看平台节奏
   - 番茄向强反馈：前三章都必须各有一个结果变化
   - 起点向慢热厚重：第一章可稳，但第二章必须显局，第三章必须挂长线钩子

若三轮判断冲突，按以下优先级取舍：

`开篇主路线 > 核心卖点 > 平台节奏`

## 路线展开表

### 1. 危机直砸型

- 第 1 章先立：眼前异常和立即危险
- 第 2 章必须推进：主角第一次主动应对，异常规则露一角
- 第 3 章必须承诺：更大异常或更深真相压下来
- 第 3 章章尾优先：`信息反转型` 或 `危机压顶型`

不要写成：

- 三章都只是在害怕
- 异常很多，但主角没真正做判断

---

### 2. 普通人入局型

- 第 1 章先立：主角是谁、缺什么、为什么会被命运推到入口前
- 第 2 章必须推进：主角做第一次主动选择，真的进局
- 第 3 章必须承诺：这条成长线以后会怎么变强、怎么翻身、怎么失去
- 第 3 章章尾优先：`欲望升级型` 或 `资源争夺型`

不要写成：

- 三章都在日常
- 入局机会出来了，主角还在原地犹豫

---

### 3. 气质压场型

- 第 1 章先立：叙事气味、人物姿态、世界压力
- 第 2 章必须推进：当前局面显形，不能只有文气
- 第 3 章必须承诺：阵营、路线或更大局势轮廓出现
- 第 3 章章尾优先：`关系突变型` 或 `信息反转型`

不要写成：

- 三章都在端着
- 读者只觉得有味道，但不知道在看什么

---

### 4. 热血反压型

- 第 1 章先立：屈辱、压迫来源、翻身欲望
- 第 2 章必须推进：第一次反手、第一次证明、第一次被重新看见
- 第 3 章必须承诺：升级路径、资源入口、真正对手轮廓
- 第 3 章章尾优先：`资源争夺型` 或 `危机压顶型`

不要写成：

- 三章都在受气
- 压迫有了，但读者还看不见翻身轮廓

---

### 5. 权谋文气型

- 第 1 章先立：人物不简单、背后有局
- 第 2 章必须推进：一层局面被看见，利益关系露出边角
- 第 3 章必须承诺：站队、身份、暗线或权力结构开始咬合
- 第 3 章章尾优先：`选择逼近型` 或 `信息反转型`

不要写成：

- 三章都在故弄玄虚
- 只有大词，没有局势交换

---

### 6. 重生囤货型

- 第 1 章先立：上一世损失、末世时限、为什么现在必须囤
- 第 2 章必须推进：囤货行动真正开始，资源与时间压力可见
- 第 3 章必须承诺：末世后的生存线、敌我线或据点线露轮廓
- 第 3 章章尾优先：`危机压顶型` 或 `资源争夺型`

不要写成：

- 三章都在列购物清单
- 囤了很多，但看不出生存主线

---

### 7. 系统囤货型

- 第 1 章先立：系统 / 空间可用，主角立刻拿到第一波反馈
- 第 2 章必须推进：囤货收益兑现到现实局面
- 第 3 章必须承诺：更大生存目标或更高资源目标成立
- 第 3 章章尾优先：`结果未揭晓型` 或 `危机压顶型`

不要写成：

- 三章都在播报系统
- 爽点有了，但世界没动

---

### 8. 长生慢发育型

- 第 1 章先立：为什么必须稳、稳的价值是什么
- 第 2 章必须推进：主角稳住第一层安全边际
- 第 3 章必须承诺：长期发育路线、寿命收益或资源积累方式
- 第 3 章章尾优先：`欲望升级型` 或 `资源争夺型`

不要写成：

- 三章都在讲道理
- 主角只会苟，不会做事

---

### 9. 趋吉避凶型

- 第 1 章先立：死局感与主角的避险能力
- 第 2 章必须推进：避险能力第一次有效证明
- 第 3 章必须承诺：风险管理如何成为主线优势
- 第 3 章章尾优先：`选择逼近型` 或 `信息反转型`

不要写成：

- 只说能避险，不给一次成功案例
- 主角一直观望，三章都不下场

---

### 10. 伪装潜伏型

- 第 1 章先立：为什么暴露即死
- 第 2 章必须推进：主角第一次靠伪装稳住局面
- 第 3 章必须承诺：潜伏成长路线与长期高压环境
- 第 3 章章尾优先：`危机压顶型` 或 `关系突变型`

不要写成：

- 只说低调，不写具体藏法
- 潜伏感有了，但环境威胁不成立

---

## 三章快检

写完开书方案后，快速检查：

1. 第 1 章是不是在抓入场，而不是先解释
2. 第 2 章是不是让主角真的开始行动
3. 第 3 章是不是已经给出长期承诺，而不是只做重复小高潮
4. 三章是不是各自都有结果变化或局面变化
5. 第 3 章章尾是不是能把读者稳稳拖进第 4 章

若第 2-3 条答不上来，说明你只是选了开篇路线，还没真正搭出黄金三章。
