# 黄金三章路线速查卡

用途：
- `/一键开书` 时把第一章路线扩成前三章功能链
- 第一章能抓人，但第二三章容易散时快速校正

原则：
- 第 1 章抓入场
- 第 2 章让主角开始行动
- 第 3 章给长期承诺并挂强钩子

## 三个必答问题

1. 第 1 章已经立住了什么：危机 / 处境 / 气质 / 屈辱 / 大局 / 生存窗口 / 苟道规则
2. 第 2 章最该补什么：主动性 / 新阻力 / 新规则 / 新机会 / 新关系 / 新代价
3. 第 3 章必须让读者看到什么：后面怎么爽 / 怎么斗 / 怎么活 / 怎么苟

## 自动推荐

- `危机直砸型`：异常 -> 应对 -> 更大异常
- `普通人入局型`：处境 -> 入局 -> 路线承诺
- `气质压场型`：气质与处境 -> 局面显形 -> 阵营与长期局
- `热血反压型`：受压 -> 第一次反手 -> 升级线承诺
- `权谋文气型`：人不简单 -> 局不简单 -> 站队与暗线
- `重生囤货型`：末世将至 -> 抢时间囤货 -> 生存线与敌我线
- `系统囤货型`：系统可用 -> 囤货见收益 -> 更大生存目标
- `长生慢发育型`：必须稳 -> 稳住第一步 -> 长期发育承诺
- `趋吉避凶型`：死局感 -> 避险验证 -> 风险管理主线
- `伪装潜伏型`：不能暴露 -> 先稳身份 -> 潜伏成长路线

冲突时按：`开篇主路线 > 核心卖点 > 平台节奏`

## 三章落地卡

### 危机直砸型
- 第 1 章：异常与危险
- 第 2 章：第一次主动应对，规则露一角
- 第 3 章：更大异常或更深真相压下

### 普通人入局型
- 第 1 章：主角是谁、缺什么、入口在哪里
- 第 2 章：做第一次主动选择，真的进局
- 第 3 章：给成长线承诺和阶段目标

### 气质压场型
- 第 1 章：立气味、姿态、压力
- 第 2 章：让局面显形，不能只剩文气
- 第 3 章：阵营、路线或更大局势露轮廓

### 热血反压型
- 第 1 章：受压与翻身欲
- 第 2 章：第一次反手或第一次证明
- 第 3 章：升级路径、资源入口、真正对手轮廓

### 权谋文气型
- 第 1 章：人不简单、背后有局
- 第 2 章：利益关系露边角
- 第 3 章：站队、身份、暗线开始咬合

### 重生囤货型
- 第 1 章：上一世损失和时限
- 第 2 章：囤货行动启动，资源和时间压力可见
- 第 3 章：生存线、敌我线或据点线露轮廓

### 系统囤货型
- 第 1 章：系统可用，先拿第一波反馈
- 第 2 章：收益兑现到现实局面
- 第 3 章：更大生存目标或资源目标成立

### 长生慢发育型
- 第 1 章：为什么必须稳
- 第 2 章：稳住第一层安全边际
- 第 3 章：长期发育路线成立

### 趋吉避凶型
- 第 1 章：死局感与避险能力
- 第 2 章：能力第一次有效证明
- 第 3 章：风险管理成为主线优势

### 伪装潜伏型
- 第 1 章：暴露即死
- 第 2 章：第一次靠伪装稳住局面
- 第 3 章：潜伏成长路线与长期高压环境成立

## 三章快检

- 第 1 章是不是先抓入场，而不是先解释
- 第 2 章是不是让主角真的开始行动
- 第 3 章是不是给了长期承诺，而不是重复小高潮
- 三章是否都有结果变化或局面变化
