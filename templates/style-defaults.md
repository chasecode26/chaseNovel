## 玄幻/修仙模板
```markdown
## 基础档案
- 书名：
- 题材：玄幻/修仙
- 平台目标：
- 叙事视角：
- 主叙述人称：
- 主风格标签：升级、机缘、反压制、力量兑现
- 辅风格标签：

## 题材口吻基线
- 节奏：中短句、强节拍
- 对话占比：35%-50%
- 允许的旁白浓度：中
- 默认情绪强度：中高
- 适合的句式节拍：短句压阵，中句承接，关键反馈短促落地
- 禁止出现的通病：大段讲设定、升级无反馈、战斗只有名词没有结果、威压全靠空话

## 单书风格DNA
- 情绪曲线：压抑→蓄力→突破→反馈
- 核心卖点：升级、机缘、反压制、力量兑现
- 句式特征：句子利落，关键突破节点要短促有力
- 语言风格：可以冷硬，但不能悬浮
- 情绪表达方式：多通过异象、代价、他人反馈体现
- 场景展开顺序偏好：异常/阻力 → 试探/交锋 → 反制 → 反馈
- 叙述距离：中近距离
- 常用感知通道：视觉、身体负荷、动作受阻、力量反馈

## 旁白规则
- 功能旁白保留条件：交代境界、规则、代价时必须与当前冲突绑定
- 视角内旁白偏好：以主角对力量差距、机会与代价的判断为主
- 作者式旁白禁区：直接宣布“天地变色般的压迫”“命运的气息”等
- 允许直接下判断的场景：大转折后的短收束
- 不允许直接下判断的场景：战斗中段、试探阶段、铺垫阶段

## 气氛渲染规则
- 压迫感优先写法：异象 -> 身体负荷 -> 动作迟滞 -> 强弱差距
- 暧昧感优先写法：克制互动，不抢主线
- 危险感优先写法：敌意、杀机、资源争夺与后果
- 悬疑感优先写法：规则缺口、遗迹异常、旧线索呼应
- 禁止使用的空泛渲染词：宿命、窒息、无边威压等连续堆叠
- 出现抽象判断时的替换策略：改写成异象、反应、结果

## 句式要求
- 句子利落，关键突破节点要短促有力
- 大战前少解释，多用动作与结果带信息

## 场景偏好
- 宗门、秘境、擂台、遗迹、资源争夺地

## 角色表达偏好
- 强者有压迫感，弱者有欲望与不甘

## 爽点/情绪点触发方式
- 卡境突破、越级反杀、资源到手、众目反转

## 章尾钩子偏好
- 更大机缘、更强敌人、身份暴露、境界异动

## 负向锚点（去AI化红线）
- 禁止句式：空泛判词、连续三句以上“威压如何如何”
- 慎用句式：大段世界观总结
- 禁止词汇：宿命、苍茫、浩瀚、无尽等连续堆叠
- 慎用词汇：压抑、沉重、冰冷
- 高频重复预警词：威压、灵压、气息、杀意
- 替代策略：先写异象、再写后果、最后再收束
```

## 都市/系统流模板
```markdown
## 基础档案
- 书名：
- 题材：都市/系统流
- 平台目标：
- 叙事视角：
- 主叙述人称：
- 主风格标签：系统奖励、现实逆袭、资源碾压
- 辅风格标签：

## 题材口吻基线
- 节奏：快节奏、即时反馈
- 对话占比：40%-60%
- 允许的旁白浓度：低
- 默认情绪强度：中高
- 适合的句式节拍：短句快收，信息快给快落
- 禁止出现的通病：系统播报过长、铺垫太久不兑现、配角全成工具人、作者腔太重

## 单书风格DNA
- 情绪曲线：受压→触发→打脸→收益
- 核心卖点：系统奖励、现实逆袭、资源碾压
- 句式特征：短句快反应
- 语言风格：口语、直接、少装腔
- 情绪表达方式：靠反馈与反应，不靠抒情总结
- 场景展开顺序偏好：压力 -> 触发 -> 执行 -> 反馈 -> 更大目标
- 叙述距离：近距离
- 常用感知通道：对话、现实反应、系统反馈、现场结果

## 旁白规则
- 功能旁白保留条件：交代系统规则时必须压缩
- 视角内旁白偏好：主角盘算、判断、临场反应
- 作者式旁白禁区：替读者宣布“这就是逆袭的快感”
- 允许直接下判断的场景：打脸结果已出后的收束句
- 不允许直接下判断的场景：冲突尚未落地时

## 气氛渲染规则
- 压迫感优先写法：现实压力、人际落差、任务惩罚
- 暧昧感优先写法：话里有话、表面正常实际越界
- 危险感优先写法：现实代价、舆论风险、任务失败后果
- 悬疑感优先写法：系统隐藏条件、身份信息差
- 禁止使用的空泛渲染词：命运、宿命、深渊、无边黑暗等作者腔大词
- 出现抽象判断时的替换策略：改写成现场反馈与可见变化

## 句式要求
- 多短句，少空泛抒情
- 信息要快给快收，最好当章见反馈

## 场景偏好
- 公司、饭局、商业现场、公开场合、书房、军府

## 角色表达偏好
- 说话带现实感，冲突要直接

## 爽点/情绪点触发方式
- 打脸、反转、奖励揭晓、身份翻盘

## 章尾钩子偏好
- 新任务、大奖励、关键人物到场、舆论变化

## 负向锚点（去AI化红线）
- 禁止句式：大段系统播报后再大段作者总结
- 慎用句式：假装深沉的旁白判词
- 禁止词汇：宿命、窒息、冰冷、无边深渊
- 慎用词汇：压迫、阴沉、沉重
- 高频重复预警词：系统、奖励、震惊、脸色一变
- 替代策略：压缩播报，强调动作和反馈
```

## 种田文模板
```markdown
## 主风格
- 题材：种田文
- 节奏：舒缓中带持续小推进
- 情绪曲线：困顿→积累→改善→期待
- 对话占比：30%-45%
- 核心卖点：经营感、生活感、关系慢热、稳定成长
- 禁忌：流水账、细节堆砌却没有变化、生活有温度但没有目标

## 句式要求
- 中句为主，保留生活细节
- 少喊口号，多写具体改善

## 场景偏好
- 村庄、家宅、田地、集市、作坊、厨房

## 角色表达偏好
- 关系变化细水长流，情绪不宜过炸

## 爽点/情绪点触发方式
- 第一桶金、生活改善、人际回暖、资源积累

## 章尾钩子偏好
- 新资源、新订单、新邻里关系、新困难
```

## 末世模板
```markdown
## 主风格
- 题材：末世
- 节奏：短句碎拍，压迫感强
- 情绪曲线：紧张→爆发→喘息→新危机
- 对话占比：25%-40%
- 核心卖点：生存压迫、异能成长、资源争夺、团队抉择
- 禁忌：一直逃命不升级、危机同质化、只写残酷不写目标

## 句式要求
- 节奏快，信息分批释放
- 感官描写要直接，不绕

## 场景偏好
- 废墟、据点、医院、地铁、封锁区、野外补给点

## 角色表达偏好
- 每次分歧都尽量带生存代价

## 爽点/情绪点触发方式
- 极限逃生、异能进化、资源抢夺、队伍翻盘

## 章尾钩子偏好
- 尸潮、据点失守、队友异变、能力失控
```

## 历史/权谋模板
```markdown
## 主风格
- 题材：历史/权谋
- 节奏：稳推进，节点爆发
- 情绪曲线：受制→借势→破局→立威
- 对话占比：40%-55%
- 核心卖点：布局、借势、阵营博弈、上位翻盘
- 禁忌：人物过多记不住、全靠解释局势、只谈谋略不出结果、旁白判词连用盖过场面

## 句式要求
- 多用含义明确的对白和动作体现立场
- 大段政治分析要拆进场景
- 旁白烘托可以有，但只在关键转折点落重句，不连续替读者下判断
- 边关争霸线优先写军令、急报、甲胄、粮道、站位与兵心，再补情绪性评语

## 场景偏好
- 朝堂、家宴、书房密谈、边地急报、军府、城头、权力交换现场

## 角色表达偏好
- 说话留余地，但立场要鲜明
- 上位者压人要靠礼法与资源，武将压人要靠军功、军纪与边地现实，不靠同一种狠话反复抬气氛

## 爽点/情绪点触发方式
- 借刀杀人、局中局、临场翻盘、上位者认可或忌惮

## 章尾钩子偏好
- 密旨、站队变化、盟友背刺、朝堂风向突变
```

若主线明显偏“边关争霸 / 男频爽文”，再补读 `templates/substyles/bianguan-zhengba-nanpin.md`，把军权兑现、边地现实、朝令掣肘和男频爽点单独拉高。
若平台目标明确偏番茄、章节需要更强抓人和反馈密度，再补读 `templates/substyles/fanqie-kuai-fankui-shuangdian.md`。
若主线里有克制、慢热、互相递刀的关系线，再补读 `templates/substyles/manre-gongmou-ganqingxian.md`。
