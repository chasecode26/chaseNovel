# 角色说话差分表

> Primary owner: `style`
> Primary consumer: `writer`
> 用途：给 Writer 一份最短可执行的角色声口差分骨架。

## 每个角色最少回答

- 身份 / 关系位置：
- 性格底色：
- 当前诉求：
- 当前最怕失去什么：
- 平时句长：短 / 中 / 长
- 说话速度：快 / 稳 / 慢
- 常用措辞：
- 常用动作伴随：
- 说话目的优先级：争利益 / 自保 / 压人 / 试探 / 传情 / 推责任
- 压力上来时会怎么变：
- 他绝不会说的话：
- 同一件事，他会怎么说：

## 对话撞车快检

- 两个角色会不会都用同类判断句
- 两个角色会不会都爱解释
- 两个角色在压力下是不是还说得一样整齐
- 哪些台词只是搬信息，不像在争利益 / 压位置 / 自保
- 若撞车，优先拉开什么：句长 / 语速 / 词味 / 目的 / 动作伴随

## 使用规则

- 先拉开句长、语速、措辞，再谈“文艺感”。
- 角色区分优先靠身份、性格、欲望、压力，不靠怪口头禅。
- 同一句意思，不同角色默认要说成不同样子。
- 如果一句台词换个名字谁都能说，这句通常还没写到位。
- 若需要整本书声音真相源，回看 `voice.md`。
