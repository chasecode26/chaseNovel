# 角色说话差分表

用途：

- 给 `StyleDialogue` 一个最短可执行的角色说话区分骨架
- 避免人物都用同一种句长、同一种判断语、同一种情绪爆发方式
- 强制把“身份、性格、压力状态”落成不同声口，而不是统一修成一种顺嘴话

```markdown
# 角色说话差分表

## 角色 1
- 身份 / 关系位置：
- 性格底色：
- 平时句长：短 / 中 / 长
- 说话速度：快 / 稳 / 慢
- 常用措辞：
- 常用动作伴随：
- 争利益时更像怎么说：
- 心虚时更像怎么说：
- 压人 / 自保时先做什么：
- 情绪上来时会怎么变：
- 他绝不会说的话：
- 他最像哪种气味：冷 / 刚 / 软 / 绕 / 稳 / 轻 / 脏
- 同一件事，他会怎么说：

## 角色 2
- 身份 / 关系位置：
- 性格底色：
- 平时句长：
- 说话速度：
- 常用措辞：
- 常用动作伴随：
- 争利益时更像怎么说：
- 心虚时更像怎么说：
- 压人 / 自保时先做什么：
- 情绪上来时会怎么变：
- 他绝不会说的话：
- 他最像哪种气味：
- 同一件事，他会怎么说：

## 对话撞车快检
- 两个角色会不会都用同类判断句：
- 两个角色会不会都爱解释：
- 两个角色在压力下是不是还说得一样整齐：
- 哪些台词只是搬信息，不像在争利益 / 压位置 / 自保：
- 若撞车，优先拉开什么：
```

使用规则：

- 先拉开句长、语速、措辞，再谈“文艺感”
- 角色区分优先靠身份、性格、欲望、压力，不靠奇怪口头禅
- 对白先承担利益冲突、试探、压制、退让、遮掩，再承担设定说明
- 同一句意思，不同角色默认要说成不同样子：
  - 武将更可能说得直、硬、带火气，先落输赢、伤亡、脸面
  - 文臣更可能先卡礼法、责任、后果、名分
  - 老兵更可能先落到守不守得住、死人、粮、伤
  - 世家子、谋士、掌柜、妇人、少年，不该都像一个作者在发言
- 人物被逼急时，先问他会不会变糙、变短、变冲，还是反而更收、更阴、更绕
- 如果一句台词换个名字谁都能说，这句通常还没写到位
- 如果一个角色删掉名字还能认出来，差分才算成立

### 速用示例

- 同一事实：宗门快守不住了
  - 武将口：`这一仗再接不住，山门都得让人踩穿。`
  - 文臣口：`三日后的裁定若失手，青岚宗连山门名分都保不住。`
  - 老兵口：`护山阵快见底了，库房那点丹也顶不了多久。`

- 同一事实：打赢这一仗能压住敌人几年
  - 概括写法：`这不是一场胜，这是把他们三年那口气都给掐断了。`
  - 更像军中人：`这一仗打完，北朔那帮狗东西往后几年都别想在咱们面前直起腰。`
