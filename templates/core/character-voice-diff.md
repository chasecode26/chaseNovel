# 角色说话差分表

用途：

- 给 Writer 一个最短可执行的角色声口骨架
- 避免人物都用同一种句长、同一种判断语、同一种情绪爆发方式
- 强制把“身份、性格、欲望、压力状态”落成不同声口，而不是统一修成一种顺嘴话

```markdown
# 角色说话差分表

## 角色 1
- 身份 / 关系位置：
- 性格底色：
- 当前诉求：
- 当前最怕失去什么：
- 平时句长：短 / 中 / 长
- 说话速度：快 / 稳 / 慢
- 常用措辞：
- 常用动作伴随：
- 说话目的优先级：争利益 / 自保 / 压人 / 试探 / 传情 / 推责任

- 两个角色会不会都用同类判断句：

- 若撞车，优先拉开什么：句长 / 语速 / 词味 / 目的 / 动作伴随
```

使用规则：

- 先拉开句长、语速、措辞，再谈“文艺感”。
- 角色区分优先靠身份、性格、欲望、压力，不靠怪口头禅。
- 对白先承担利益冲突、试探、压制、退让、遮掩，再承担设定说明。
- 同一句意思，不同角色默认要说成不同样子。
- 人物被逼急时，先问他会不会变糙、变短、变冲，还是反而更收、更阴、更绕。
- 如果一句台词换个名字谁都能说，这句通常还没写到位。
- 如果一个角色删掉名字还能认出来，差分才算成立。
