## 使用规则

- `assets/common/style-defaults.md` provides genre defaults; this file keeps per-book style choices.
- 先定读者体验与口吻，再补角色说话方式。
- 风格不能推翻章节功能，只能强化表现。
- 语言发滑时，先删判词和空泛情绪，再补动作、站位、结果。
- 默认写明白，不绕，不装深沉；只有悬疑关键点才允许少量留白。
- 如果一段已经有一刀，就不要再追加第二刀、第三刀。
