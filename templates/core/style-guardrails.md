# 风格护栏

> 写前必读。遇到“句子像有劲但读者看不懂”“政治逻辑没摊开”“台词发硬”时优先回看。

## 默认表达

- 默认清晰可读，不默认写成统一口水化。
- 先写谁在做事，再写为什么做，再写结果变了什么。
- 普通章节不故作高深，不靠谜语句撑气氛。
- 问句出来后，下一句必须答上，不准只顶一句有气势的话。
- 军令和判断句必须说透，不准只留半截黑话或术语。

## 判断句顺序

1. 先报事实。
2. 再报人物判断。
3. 最后报具体后果。

## 禁写

- “不是试探，是要继续压”
- “局势更坏了”
- “气息一下紧了”
- “那个人”“那件事”“某种真相”式普通信息绕写
- 只给术语，不给动作和后果
- “把咱们眼睛全拽过去”“那边才有鬼”这种看着有劲、实际没说透的战术黑话
- “这才是真刀”“真正要命的是”“不是……而是……”这种作者提炼腔
- 问“怎么办/为什么/什么意思”后，只答“对”“我知道”“回去”“兄弟？”这种半截短句
- 用“冷、稳、狠”当借口，把话说得太短，最后读者还得自己补意思
- “说到底”“本质上”“某种意义上”“答案很简单”“显然”“毫无疑问”这类作者总结句
- “压迫感”“拉扯”“博弈”“张力”“情绪价值”“信息差”这类分析词直接顶正文
- 上章说好的时间、地点、人数、伤势、兵器、命令，下一章不交代原因就改口

## 最小连续性先查

写章前先只核对四件事，不对齐就先修 `state.md / timeline.md / 章卡`，不要直接起稿：

1. 时间承诺有没有对上。
2. 人物、物资、信使、援手的位置和到达时间能不能对上。
3. 上章后果现在还在不在。
4. `state.md / timeline.md` 里有没有把上一章已经钉死的信息写清。

## 写章前硬核对清单

起稿前必须先过一遍，任何一项答不清，就先补 `state.md / timeline.md / 章卡`，不要硬写。

### 1. 时间核对
- 现在是立刻接上章，还是已经过了一夜/半天/几天。
- 上章说的“今晚、明早、明天、三日后”有没有真的对上。
- 赶路、养伤、调兵、送信，时间够不够。

### 2. 人数与在场人物核对
- 这一场里到底有几个人。
- 谁已经离开，谁中途加入，谁还没赶到。
- 上章写死的三个人，这章不能无故变成十个人。

### 3. 伤势与体力核对
- 谁受了什么伤，现在能做到什么，做不到什么。
- 若伤势变轻、体力恢复、能继续打，必须有恢复时间、用药、硬撑代价或他人接手。

### 4. 资源与兵器核对
- 现在手里还有什么：兵器、药、钱、马、粮、人手、情报。
- 上章说缺的东西，这章不能默认突然有了；若补上了，要写清谁送来的、什么时候送到。

### 5. 消息与知情边界核对
- 谁已经知道，谁还不知道。
- 新消息是谁带来的，什么时候到的，够不够快。
- 角色不能知道自己没听见、没看到、没推出来的事。

### 6. 命令与承诺核对
- 上章已经说出口的命令，现在是谁在办。
- 上章答应要做的事，这章有没有接上。
- 如果还没兑现，必须写清卡在哪一步，不准直接跳过。

### 7. 战力与场景常识核对
- 这个人现在能打几个，靠什么打，为什么打得过。
- 若局面突然变强，必须补援手、地形、埋伏、兵器、时机。
- 若局面突然变弱，也要补清楚为什么撑不住。

### 8. 空间与位置核对
- 这一章开场时，人分别在哪，东西分别在哪。
- 谁来得及赶到，谁根本来不及到场。
- 若人物、物资、援手、信使换了位置，必须写清怎么过去的、花了多久。

### 9. 后果延续核对
- 上一章的夜袭、翻脸、受伤、断粮、透支、暴露，这一章后果还在不在。
- 如果后果变轻，必须补休整、补给、遮掩、安抚或转移代价。
- 如果后果已经解决，必须写清是谁解决的、怎么解决的、代价落到谁头上。

### 10. 本章推进上限核对
- 这一章最多只能推进到哪一步。
- 哪些结果这章能写，哪些结果还不能提前兑现。
- 如果一章里把铺垫、转折、结算全写完，先怀疑是不是推太快了。

## 最小硬核对模板

```markdown

- 上章后果现在是否仍在：

```

## 改写偏好

- 能直接报有效信息，就不要写反衬句。
- 能写“箭和伤药什么时候送到”，就不要写“箭车何时出关”。
- 能写“守口子的那段矮墙”，就不要写“后墙”。
- 能写“桌案上摊着的是关防图、物资名录、伤营名录”，就不要先写“不是礼单”。
- 能把后果讲清楚，就不要先说“这是刀”“这是局”“这是真正要命的地方”。
- 能把命令链写全，就不要只写“去办”“盯住”“给我查”。
- 能一句说透，就不要拆成两句半截硬句故作压迫。

## 本项目复盘后新增护栏

- 历史权谋戏不能再写成“作者在分析局势”，要写成“人物在处理局势”。
- 问话后的第一反应，优先给结论，不要先给态度。
- 朝堂、军议、后宅账房三类戏，必须写出谁最先吃亏、谁最先受制、谁最先要补后账。
- 如果一句台词删掉后，信息没少，只剩“味道”，那句就删。

## 战争 / 权谋补充

- 皇帝、朝臣、边将都只能按自己知道的信息判断。
- 政治逻辑要写清“他想拿到什么结果”，不能只写“他很狠”。
- 边军归属要写成“认人不认印”，不是只写一枚印。
- 敌情、军报、推理句必须直接说透：对方要干什么、从哪动手、你凭什么这么判断。
- 能写“正门只是幌子，他们想趁黑从东坡摸上来”，就不要写“主口不是主杀招，东坡那边有鬼”。
- 命令一旦落下，后面立刻补：谁去、先查哪、多久回报、慢一步会怎样。
