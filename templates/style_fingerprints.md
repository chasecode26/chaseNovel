# 风格指纹库 (Style Fingerprints)

本库用于在 `/定风格` 阶段，为一本书指定更具体的叙述气质。

它不是“审美标签墙”，而是可直接落进 `style.md` / `voice.md` 的执行型风格指纹。
先选最贴近当前书的 `1` 条主指纹；必要时最多叠 `1` 条辅指纹，不要混成大杂烩。

使用前先记住两条：

- 风格指纹不能推翻题材推进器，只能决定“这本书怎么说”。
- 清晰可读不等于口水化；文气成立也不等于故作高深。

## 1. 利落现实档
**适用题材**：都市系统、现实逆袭、职场博弈、快反馈男频

- **叙述距离**：贴近主角当下得失与现场反馈，少抒情回望。
- **句式节拍**：短中句为主，关键结果句要落得快，但不写成流水播报。
- **对白规则**：话要短、准、带功能；试探和讥讽可以有，但不能写成谜语人互丢黑话。
- **情绪推进**：靠困境、执行、反馈、外界反应四步推进，不靠围观群众替主角喊爽。
- **引文边界**：默认不用诗句典故；若用，只能是人物身份自然会说出的东西。
- **易错点**：容易滑成纯口水、纯播报、纯打脸模板。

## 2. 克制机锋档
**适用题材**：历史权谋、朝堂试探、古风谋局、慢热群像

- **叙述距离**：第三人称限知，贴当前持牌者或承压者，不乱切全知。
- **句式节拍**：中句稳推，节点处才下重句；平时靠节奏和站位积压，不靠判词硬压。
- **对白规则**：允许潜台词、礼法、留手、话里夹刀，但读者必须判断得出谁在试探、谁在施压、谁在让步。
- **情绪推进**：靠礼数失衡、称呼变化、军令措辞、物件落点、沉默时长推进。
- **引文边界**：可低频到中频用典，但必须符合身份和场景，不能拿典故代替局势。
- **易错点**：容易写成文绉绉空转，或全员都像同一位老练作者。

## 3. 苍凉高压档
**适用题材**：边关战争、末世压迫、盗墓奇诡、生存逃亡

- **叙述距离**：优先贴眼前危险与身体代价，不拉远讲宏大立意。
- **句式节拍**：短句与中句交替，压迫段可碎拍，收束段可稍拉长留余波。
- **对白规则**：对白要服从危机，不宜太满；一句话最好同时带结果、判断或资源信息。
- **情绪推进**：靠空间限制、时间压力、补给变化、体力流失、错误判断推进。
- **引文边界**：默认慎用；只有碑刻、旧信、遗言、题壁等自然点位可用。
- **易错点**：容易只剩“肃杀、压抑、苍凉”这些气氛词，没有硬支点。

## 4. 群像悲壮档
**适用题材**：战争群像、历史高武、长线关系网、宿命型卷段

- **叙述距离**：允许在卷级切不同焦点人物，但单场景仍锁单焦点。
- **句式节拍**：整体稳，转折处快；大悲大壮最好用克制句收，不用滥情重锤。
- **对白规则**：人物要有鲜明身份差、立场差、旧账差，不能只靠“金句”区分人。
- **情绪推进**：靠旧账回收、关系断裂、牺牲后果、阵营改写积累情绪。
- **引文边界**：可少量使用，但必须让位于人物命运和局面变化。
- **易错点**：容易沉迷名句感、意难平感，结果真正推进不足。

## 5. 紧绷破局档
**适用题材**：限时破案、制度夹缝求生、微观权谋、多线并发章节

- **叙述距离**：紧贴当前问题链，优先给线索、阻力、错误判断、修正动作。
- **句式节拍**：信息密，但句子不要打结；一段只承一层推进。
- **对白规则**：对白重交换、试探、推诿、压价、问责，废话越少越好。
- **情绪推进**：靠时间压缩、规则卡死、链条断裂、旧判断被推翻推进。
- **引文边界**：默认极低；风格重在机锋和链路，不在文饰。
- **易错点**：容易写成纯信息墙，或者全员像审讯机器人。

## 怎么选

- 要快反馈，但不想变成统一番茄腔：先选 `利落现实档`
- 要权谋、礼法、机锋，但不想写成老气空文：先选 `克制机锋档`
- 要压迫、生存、边地、墓下、危机感：先选 `苍凉高压档`
- 要关系网、宿命感、配角后劲：先选 `群像悲壮档`
- 要破局、限时、线索、规则对冲：先选 `紧绷破局档`

## 叠加规则

- 主指纹决定整体声音。
- 辅指纹只补一个维度，比如：
  - `利落现实档` + `紧绷破局档`
  - `克制机锋档` + `群像悲壮档`
  - `苍凉高压档` + `紧绷破局档`
- 如果两个指纹对句式、对白、引文边界要求相反，优先服从题材与当前卷段功能。
