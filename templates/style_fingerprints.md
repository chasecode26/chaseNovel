# 风格指纹库 (Style Fingerprints)

本库用于在 `/定风格` 阶段，为一本书指定更具体的叙述气质，避免只写成空泛的“克制”“热血”“高级感”。

## 1. 冷峻克制档
- **意象偏好**：深巷、冷雨、长街、灰色的穹顶、破旧的物件、不经意的叹息。
- **叙事节拍**：前置极度压抑与慢热，铺垫几近琐碎的生活细节，然后在一瞬间爆发。喜欢在暴烈的动作前加一句极其平淡的陈述。
- **对话密度与语气**：中低密度。对话大量留白，潜台词极度丰富。角色说话通常很少，喜欢用陈述句表达疑问，用句号代替问号。
- **情绪递进方式**：表面越平静，内部情绪拉扯越剧烈。极少写“他愤怒极了”，而是写“他看着落下的雨滴，把手洗得很干净，然后拔出了刀。”
- **禁忌表达**：绝对禁用浮夸的形容词（如大惊失色、恐怖如斯）、禁止大段的心理剖析。

## 2. 苍茫史诗档
- **意象偏好**：星空、青铜巨棺、苍茫大地、残破的古战旗、大漠、染血的祭坛。
- **叙事节拍**：大开大合。极其擅长宏观视角的拉抬，动辄跨越万古岁月。战斗场面如同神话史诗。
- **对话密度与语气**：中等密度。配角多用来衬托主角的不可思议。主角语录往往霸气侧漏、视死如归或带着横推万古的无敌信念。
- **情绪递进方式**：层层加码的震撼。先是一个小区域的震惊，然后扩散到大域，最后宇宙共鸣。
- **禁忌表达**：禁止小家子气的市井扭捏、禁止琐碎的日常拌嘴影响大局。

## 3. 直给热血档
- **意象偏好**：武魂、华丽的魂环、璀璨的光芒、极致的数字伤害、伙伴的羁绊。
- **叙事节拍**：即时反馈。升级如喝水，装逼打脸是常态。设定异常详尽且有数据化倾向，每升一级都有具体的新技能展示。
- **对话密度与语气**：高密度。对话极其直白，情绪全部写在脸上。兄弟情义、爱人誓言常常直接喊出来。
- **情绪递进方式**：绝境 -> 伙伴羁绊爆发/底牌掀开 -> 极其华丽的技能反杀。
- **禁忌表达**：禁止谜语人、禁止过长的文青式伤春悲秋、禁止主角长久的憋屈。

## 4. 群像悲壮档
**【最适用题材】：古代权谋、战争、高武历史**
- **意象偏好**：风雪、老酒、残剑、关外黄沙、藩王铁骑、读书人的风骨、落魄的江湖客。
- **叙事节拍**：极其注重“起承转合”中的“转”。前文可以用大段看似随意的对话铺排人物性格，而在战争或权谋爆发时，往往用轻描淡写的一两句话收尾，制造极强的余韵和悲壮感。
- **对话密度与语气**：中高密度。对话极具个性化，喜欢借人物之口说出极具哲理或豪气的“金句”。庙堂之争不靠大声密谋，而是靠倒酒、品茶、下雪时的轻言细语定人生死。
- **情绪递进方式**：小人物的悲凉与大人物的算计交织。极度擅长写“群像”，死一个配角往往能引发全书最大的情绪高潮，讲究“意难平”。
- **禁忌表达**：禁止纯粹无脑的“为了坏而坏”的反派、禁止干瘪的数据化战力对比、禁止将战争写成简单的数字消耗，而要写出“白骨露于野”的厚重。

## 5. 考据机锋档
**【最适用题材】：朝堂悬疑、微观权谋、限时破局**
- **意象偏好**：卷宗、漏斗/沙漏、坊市暗巷、公文流转、复杂的古代地名与官职、底层官吏的算盘。
- **叙事节拍**：像滴答作响的定时炸弹。极度紧凑，多线并发。经常把一个微小的线索（如一笔对不上的账目、一个不起眼的驿站）无限放大，最终牵扯出惊天大局。
- **对话密度与语气**：极高密度的信息交换。对话中充满了专业的古代官僚术语、推诿扯皮与暗藏机锋。没有一句废话，全是在试探底线和交换利益。
- **情绪递进方式**：通过剥丝抽茧的解谜和层层加码的“时间压力”来推高情绪。主角永远在和时间赛跑，在规则的夹缝中求生。
- **禁忌表达**：禁止主角靠全知全能的“上帝视角”破局，只能靠严密的逻辑和现有的线索；禁止用魔法或超自然力量解决现实的政治死局。
