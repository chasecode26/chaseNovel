# 题材索引

用途：
- 作为题材导航页，先判断该读哪一条细分模板
- 不在这里重复展开题材正文

## 使用规则

- 先定 `1` 个主题材，再决定要不要叠 `1` 个副题材
- 节奏冲突时，优先服从主题材
- 特殊风格或大纲目标可覆盖通用快节奏公式

## 混合题材快判

- 都市 / 系统流 + 悬疑：都市为主，悬疑负责信息差
- 玄幻 / 修仙 + 历史 / 权谋：看主卖点，升级成长优先玄幻，站队博弈优先权谋
- 末世 + 种田：末世为主，种田负责囤货、经营、据点建设
- 历史 / 权谋 + 盗墓 / 摸金：看主卖点，朝堂/站队优先历史，古墓/谜案/机关优先盗墓
