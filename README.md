# chaseNovel (白金级中文网文连载写作器)

面向中文长篇网文平台连载的终极 AI 写作引擎（基于 Claude Code Skill / Gemini）。
它不仅能写单章，更是一套集成了**商业网文方法论、风格指纹复刻、百万字大纲防崩盘管理**的专业写作流水线。

## ✨ 核心特性升级 (v2.0)

- **🖋️ 顶级作家风格指纹库 (Style Fingerprints)**：告别“AI味”，支持一键加载【猫腻风】、【烽火戏诸侯风】、【辰东风】、【唐家三少风】、【马伯庸风】，精准复刻大神经典的意象偏好、句式节奏与对白潜台词。
- **🛡️ 三级覆写防冲突协议 (Override Protocol)**：完美解决大型知识库的“人格分裂”。系统执行严格的优先级铁律：【指纹库/题材规则】 > 【大纲核心节点】 > 【通用爽文公式】。
- **📚 16大核心题材极深适配**：基于专业网文知识库，为种田、末世、修仙、权谋、克苏鲁等题材注入专属的黄金结构、写作禁忌与防毒点规范。
- **⚙️ 全生命周期大长篇管理**：
    - `power_economy_constraint.md`: 力量体系防通胀（50万字不崩盘）
    - `antagonist_design.md`: 顶级反派与配角架构（反镜像法）
    - `romance_arc.md`: 感情线独立管理（六阶段甜虐节奏追踪）
    - `ending_blueprint.md`: 终局规划（4大结局模型设计）
    - `break_writers_block.md`: 卡文急救箱（5大卡文病因靶向治疗）
- **🔪 严苛的“净身”润色流水线 (Vibe Check)**：自审脚本配合 5 层润色机制，斩断长句、去形容词、禁用“像...一样”/“不禁”等低级 AI 模板词汇。

## 🎯 基础操作

- `/一键开书`：生成包含核心卖点、冲突、阶段目标与卷节奏的开局方案。
- `/写`：根据章卡起草正文，严格遵守“先行动后信息”、“章尾必留悬念”铁律。
- `/修改` / `/续写`：定向修章、状态恢复、应对断更。
- `/防重复` / `/查承诺` / `/定风格`：随时调出辅助功能进行节奏干预。

## 📂 项目架构

- `SKILL.md`：引擎主控，存储指令路由表、三项核心铁律、读取策略。
- `references/`：通用执行工作流（写前/起草/写后/自检）、诊断规则。
- `templates/`：各类题材的专业提示词、剧情套路库、结局/战力表。
- `templates/genres/`：16类题材子库分离式加载。
- `scripts/`：自动触发的 Python 小脚本，检查章节门禁（自动检测 AI 词汇），出具周期体检报告。

## 适用场景

适合立志完成 **50 万字以上连载**、渴望跳出普通 AI 水文泥潭、需精准控制追读率与爽感节奏、追求**网文白金作家级文字质感**的文字创作者。
