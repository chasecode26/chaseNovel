# chaseNovel

面向中文长篇网文平台连载的 Claude Code Skill。

## 功能

- `/一键开书`：生成开书方案、卖点、阶段目标与卷节奏
- `/写`：按章卡推进正文，控制节奏、钩子与兑现
- `/修改` `/续写`：修章、续写、断更恢复
- 周期自检：每 5 章触发一次小自检，每 10 章触发一次大自检
- 记忆文件模板：主线计划、当前状态、人物、伏笔、承诺兑现、时间线等
- 题材支持：玄幻、修仙、都市系统、种田、末世、甜宠、悬疑、权谋等

## 目录结构

- `SKILL.md`：主技能文件
- `references/`：按需加载的长规则、长模板与诊断说明
- `skill.json`：技能元数据
- `scripts/`：自动判定 5/10 章自检触发的辅助脚本
- `hooks/`：Claude `PostToolUse` hook 包装器与配置片段
- `templates/`：写作模板、题材技法库、章卡示例与记忆文件骨架

## 用途

适合需要持续推进中文网文连载、控制章节功能、减少水章、维护设定一致性、管理伏笔与回收节奏的写作场景。
