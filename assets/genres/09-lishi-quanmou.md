## 开书前必须先补齐的骨架

如果是历史权谋 / 朝堂 / 边关 / 养兵夺权，开书时不能只有卖点、人设和三章路线，至少要先锁清：

- 朝堂体系：皇帝、中枢、监军、地方、军府、宗室、外戚怎么互相制衡
- 兵力布局：谁手里有兵，兵放在哪，主角能直接调哪一层
- 情报系统：边军、驿站、内廷、商路、密探分别怎么传消息，谁能截断
- 各国 / 外敌详情：邻国、边患、盟友、属部各自想要什么
- 财政粮道：军需从哪来，粮道从哪走，哪一环最容易出事
- 议题池：这本书真正会反复争的事是什么，不要只写“大局动荡”
- 反派池：至少准备朝堂、地方、军中、外敌四类反派，不要只有一个死对头

如果这些还空着，就直接写正文，后面最容易变成：

- 朝堂靠吵
- 权谋靠嘴
- 战争靠战报
- 反派靠轮流送头

## 一、 核心逻辑：利益、立场、时机的博弈

历史权谋的本质不是“谁更会说漂亮话”，而是**利益、立场、时机与资源调度的博弈**。读者追读的动力来自：

- 谁借到了势
- 谁失了先手
- 谁在礼法里动刀
- 谁把个人命运绑上了国家机器

## 二、 写法重点

1. **优势来自规则差、信息差、资源差**
   - 可以是眼界、制度理解、后勤调度、情报整合、识人用人，而不只是一句“现代思想降维打击”
   - 真正的优势不是嘴上高明，而是能改变别人可用的牌

2. **潜台词与礼法**
   - 朝堂之上，句句像关心，实际都在改筹码和站位
   - 很少有人把“我要弄死你”挂在嘴边，更多是借敬酒、送礼、旧案、祖制来敲打
   - 将门黑话、朝堂八股、市井俗语要分开写
   - 允许收着说，但读者必须判断得出谁在试探、谁在压价、谁在递刀

3. **计中计矩阵**
   - 明面进攻 A 是虚的，逼迫对手调动 B 也是虚的，最终目的是收割 C
   - 读者必须有"原来这一步十天前就下了"的恍然大悟感
- 但不能滥用“早就布好局”当万能解释，前提必须提前埋下

5. **默认用现代通俗人话，不装古风。**
   - 历史权谋的“味”来自礼法、站位、称呼、动作、沉默、筹码，不来自半文半白。
   - 读者看到一句话，必须马上知道谁在压谁、谁在试谁、谁先吃亏。
   - 不要靠 `看人 / 看局 / 路数 / 底子 / 那条线` 这种空词装懂。

4. **立场、利益、时机，比聪明话更重要**
   - 权谋要落到选择结果，不要只靠台词装深
   - 大局和个人命运要绑在一起
   - 文气可以有，但必须服务局势，不服务作者自我感动

## 三、 常见失误

| 失误 | 表现 | 修正方向 |
|------|------|---------|
| 空谈局势 | 全是计谋说明，没有即时危机 | 每次博弈都要明确谁赢了半步、谁失了先手 |
| 人物太多分不清 | 读者不知道谁重要 | 每章聚焦 1-2 个关键人物的立场变化 |
| 全员一个脑子说话 | 所有人说话风格一样 | 按身份写对话：朝堂八股、将门黑话、市井俗语 |
| 旁白判词过密 | 作者频繁替读者下判断 | 一场重戏最多留一两句重旁白，其余让动作和反应承担气氛 |
| 半文半白装质感 | 看着有味，实际像在端着写 | 全部拆回现代通俗大白话，让礼法、称呼、站位自己出味 |
| 木偶配角 | 所有人都配合主角的计谋 | 给对手正确的价值观和可执行计划 |
| 无视财政空谈战争 | 不写军队补给和国家财政 | 战争打的是钱粮、后勤、违令后账 |
| 只剩文气没有链路 | 写得有味，但看不出谁赢谁亏 | 每场博弈补清代价、筹码、立场变化 |

## 四、 推进技巧

- 每次博弈都要明确谁赢了半步、谁失了先手
- 权谋场面里最好有清晰代价
- 阵营变化比空谈局势更有力
- 一场重戏最多留一两句重旁白，其余让动作、物件细节和他人反应承担气氛
- 没有永远的敌人：昨天还不死不休，今天因为党争利益也能联手
- 同一卷里要拉开“朝堂机锋”“军府决断”“私下试探”三种说话方式，不要一把尺子量到底

## 五、 差例 / 好例

**差例**：
> 鼓声一响，人人都知道他这一趟凶多吉少。这不是奉召，这是押送，是朝廷要把一件旧物重新收入库房。

**好例**：
> 圣旨刚合上，传旨太监并没有把明黄绢轴递过去。
> 他只侧开半步，让出身后的羽林卫："殿下，请。"

**为什么**：差例把判断都交给旁白，气氛很重，但场面本身没有落地。好例通过递旨方式、站位和称呼变化，把"奉召还是押送"交给读者自己判断。权谋最有力的压迫，往往来自礼数失衡、军令措辞和人物反应，而不是作者连着下判词。

## 六、 章尾钩子推荐

| 类型 | 示例 |
|------|------|
| 圣意变化 | 内侍带来的不是赏赐，而是一杯御酒 |
| 盟友背刺 | 他推门进来时，手里拿着那份本该烧掉的密信 |
| 密旨到来 | 蜡丸捏碎，里面只有一行字："勿信王某" |
| 朝堂发难 | 御史出列的那一刻，他知道今天的局已经布好了 |

## 七、 节奏与句式

- **中句稳推进、节点爆发**，允许关键处收句留味，但不能写成整段发雾
- **对话占比 40%-55%**
- 朝堂、军府、边关场面先靠军令、称呼、站位、沉默和反应给压力，旁白只在关键转折补一刀

## 八、 禁忌表达

- 用极度幼稚的话对骂
- 所有人都像木偶一样配合主角的计谋
- 无视军队补给和国家财政空谈打赢大仗
- 作者频繁替读者下判断，句句都像判词
- 边关线只有肃杀和苍凉，没有军情、补给、兵心、地形这些硬支点
