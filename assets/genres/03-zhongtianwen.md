## 开书前必须先补齐的骨架

种田 / 基建文开书时，至少先锁清：

- 家里到底几口人，谁能干活，谁会拖后腿
- 地、水、种子、农具、牲口、手艺最缺哪几项
- 当前季节和未来几轮季节约束
- 税、债、粮、药、劳力哪一个最卡脖子
- 村庄 / 族老 / 乡绅 / 商路怎么卡主角
- 后面靠什么从“活下去”升到“过得越来越好”

不补这些，后面很容易只剩：

- 流水账种地
- 收成数字涨了，但生活秩序没变
- 邻里和市场都像背景板

## 一、 核心逻辑：微观积累带领宏观质变

种田文的本质是通过**资源锚定**和**循环积累**，从匮乏走向丰饶，让读者在稳定的微小进步中获得强烈安全感。核心爽点不是"突然暴富"，而是"每天都在变好"的积累感。

## 二、 写法重点

1. **资源锚定基石**
   - 尽量尽早立住一种核心底层生产资料（唯一变异灵植种子、废土净水器图纸、被遗忘的古代酿酒方）
   - 可度量进展很好用，但不必所有项目都写成面板或进度条；也可以通过工序、存量、邻里反馈、生活改善来体现积累

2. **黄金循环公式**
   - `获得资源残篇 → 艰难培育/技术改良 → 产量爆炸/品质跃升 → 财富/势力倍增 → 吸引新需求/敌人 → 进入下一轮循环`

3. **反差爽感营造**
   - 可以利用认知差、技术差、信息差、组织差形成优势，不必偷懒写成单一“现代人降维打击”
   - 用平淡甚至奢侈的日常生活方式，对比周围残酷/恶劣的生存环境

4. **过程就是爽点**
   - 不能一笔带过式"过了三年，他成为了首富"
   - 每一步改善都要写出具体动作和感受（翻土的手感、第一锅酒出甑的香气）

## 三、 常见失误

| 失误 | 表现 | 修正方向 |
|------|------|---------|
| 流水账 | 每天种地→收获→卖钱，循环重复 | 每轮循环引入新变量（新作物、新关系、新危机） |
| 太平太稳 | 只有岁月静好，没有阶段目标 | 定期在快大丰收时加入"天灾"或"恶霸觊觎" |
| 关系纸片化 | 邻居/合伙人只是工具人 | 让合作关系有动态变化（信任建立→利益冲突→重新站队） |
| 电子账本推进 | 只写数字增长，不写人的变化 | 每次资源增长必须伴随人物关系或生活秩序变化 |

## 四、 推进技巧

- 每章至少改善一件事：资源、住处、名声、关系、心态
- 小危机能增强满足感（天气、订单、邻里压力）
- 温暖感要靠具体动作，不靠空泛描述
- 大丰收后最好接新需求或新威胁，防止疲劳

## 五、 差例 / 好例

**差例**：
> 她今天去地里看了看，又回家做了饭，吃完饭以后想到以后应该会越来越好，心里很满足。

**好例**：
> 她蹲在新翻好的菜垄前，指尖沾着湿土。
> 这是她来到这里以后，第一次确认——这块地，真的是自己的了。

**为什么**：差例是概括心情，缺少具体变化。好例让"拥有感"落到一个可见结果上。种田文的满足感来自具体改善，而不是笼统自我安慰。

## 六、 章尾钩子推荐

| 类型 | 示例 |
|------|------|
| 新资源 | 后山的泉眼突然变了颜色 |
| 新订单 | 县里来了个商人，点名要她家的货 |
| 新角色进场 | 村口来了个外乡人，扛着行李在她家门前站了很久 |
| 关系试探 | 隔壁王婶今天破天荒地送了一碗鸡蛋过来 |

## 七、 节奏与句式

- **中句、偏舒缓**，段落可以稍长，营造安定感
- **对话占比 30%-45%**
- 允许缓慢、连续的小奖励（多长出一片叶子、邻居态度的轻微转变）

## 八、 禁忌表达

- "过了三年，他成为了首富"（跳过过程等于跳过爽点）
- "日子一天天好起来了"（概括代替具体改善）
- 无成本的天降资源（所有收获必须有劳动过程）
- 假平静（表面岁月静好，实际没有任何推进）
